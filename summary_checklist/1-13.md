Dưới đây là **checklist kiến thức Minna no Nihongo bài 1–13** để bạn tự soạn bài tập. Phần này chủ yếu là **mẫu câu cơ bản, trợ từ, tính từ, động từ ます形, thời gian, nơi chốn, số lượng, mong muốn và mục đích đi đâu làm gì**.

# Checklist tổng hợp Minna bài 1–13

## A. Nền tảng bắt buộc

### 1. Chữ và phát âm

* [ ] Đọc được Hiragana.
* [ ] Đọc được Katakana.
* [ ] Biết trường âm: おばさん / おばあさん.
* [ ] Biết âm ngắt っ: きて / きって.
* [ ] Biết âm ghép: きゃ、きゅ、きょ、しゃ、しゅ、しょ.
* [ ] Nhận diện được dấu kéo dài trong Katakana: コーヒー、スーパー.
* [ ] Biết cách đọc số cơ bản.
* [ ] Biết cách đọc ngày, tháng, giờ, phút cơ bản.

# B. Bài 1 — Danh từ, giới thiệu bản thân

## 1. Mẫu câu chính

* [ ] **N1はN2です** = N1 là N2.
* [ ] **N1はN2じゃありません** = N1 không phải là N2.
* [ ] **N1はN2ですか** = N1 có phải là N2 không?
* [ ] **はい、そうです** = vâng, đúng vậy.
* [ ] **いいえ、そうじゃありません** = không, không phải vậy.
* [ ] **N1もN2です** = N1 cũng là N2.
* [ ] **N1のN2** = N2 của N1 / thuộc về N1.
* [ ] **〜さん** = anh/chị/bạn… dùng sau tên người.

## 2. Kiến thức cần nắm

* [ ] Biết giới thiệu tên:
  私はアンです。
* [ ] Biết nói quốc tịch:
  私はベトナム人です。
* [ ] Biết nói nghề nghiệp:
  私は学生です。
* [ ] Biết hỏi người khác là ai / nghề gì / quốc tịch gì.
* [ ] Biết dùng **は** để nêu chủ đề.
* [ ] Biết dùng **も** với nghĩa “cũng”.
* [ ] Biết dùng **の** để chỉ sở hữu hoặc thuộc tính.

## 3. Lỗi dễ sai

* [ ] Không dùng **さん** cho chính mình.
* [ ] Không nhầm **は** với **が** ở giai đoạn này.
* [ ] Không dịch máy móc **の** chỉ là “của”; đôi khi là thuộc tính:
  日本語の先生 = giáo viên tiếng Nhật.

---

# C. Bài 2 — Đại từ chỉ vật, sở hữu

## 1. Mẫu câu chính

* [ ] **これはNです** = đây là N.
* [ ] **それはNです** = đó là N.
* [ ] **あれはNです** = kia là N.
* [ ] **これは何ですか** = đây là cái gì?
* [ ] **これはN1ですか、N2ですか** = đây là N1 hay N2?
* [ ] **これはN1のN2です** = đây là N2 của N1.
* [ ] **このN** = N này.
* [ ] **そのN** = N đó.
* [ ] **あのN** = N kia.
* [ ] **そうですか** = vậy à / thế à.

## 2. Phân biệt こ・そ・あ

* [ ] **こ** = gần người nói.
* [ ] **そ** = gần người nghe.
* [ ] **あ** = xa cả hai.
* [ ] これ / それ / あれ dùng độc lập.
* [ ] この / その / あの phải đi với danh từ.

Ví dụ:

* [ ] これは本です。
* [ ] この本は私のです。

## 3. Sở hữu

* [ ] **私の本** = sách của tôi.
* [ ] **これは私のです** = cái này là của tôi.
* [ ] Biết lược bỏ danh từ khi đã rõ nghĩa:
  私の本です → 私のです.

## 4. Lỗi dễ sai

* [ ] Không nói **このは本です**.
* [ ] Phải nói **この本は〜です**.
* [ ] Không nhầm **これ** và **この**.

---

# D. Bài 3 — Nơi chốn, địa điểm

## 1. Mẫu câu chính

* [ ] **ここはNです** = chỗ này là N.
* [ ] **そこはNです** = chỗ đó là N.
* [ ] **あそこはNです** = chỗ kia là N.
* [ ] **Nはここです** = N ở đây.
* [ ] **Nはどこですか** = N ở đâu?
* [ ] **Nはどちらですか** = N ở đâu? / phía nào? lịch sự hơn.
* [ ] **Nは国名・会社名のNです** = N thuộc nước/công ty nào.
* [ ] **これはどこのNですか** = đây là N của nước/công ty nào?
* [ ] **いくらですか** = bao nhiêu tiền?

## 2. Đại từ nơi chốn

* [ ] ここ = ở đây.
* [ ] そこ = ở đó.
* [ ] あそこ = ở kia.
* [ ] どこ = ở đâu.
* [ ] こちら = phía này / vị này / cách lịch sự.
* [ ] そちら = phía đó / vị đó.
* [ ] あちら = phía kia / vị kia.
* [ ] どちら = phía nào / vị nào.

## 3. Kiến thức cần nắm

* [ ] Hỏi địa điểm:
  トイレはどこですか。
* [ ] Nói địa điểm:
  トイレはあそこです。
* [ ] Hỏi giá:
  これはいくらですか。
* [ ] Hỏi xuất xứ:
  これはどこの時計ですか。

## 4. Lỗi dễ sai

* [ ] Không nhầm **ここ** với **これ**.
* [ ] Không nhầm **どこ** với **どれ**.
* [ ] Biết **どちら** lịch sự hơn **どこ**.

---

# E. Bài 4 — Thời gian, động từ cơ bản

## 1. Mẫu câu chính

* [ ] **今、何時ですか** = bây giờ là mấy giờ?
* [ ] **今、〜時〜分です** = bây giờ là … giờ … phút.
* [ ] **Nは〜時から〜時までです** = N từ … giờ đến … giờ.
* [ ] **〜時にVます** = làm V lúc … giờ.
* [ ] **Vます** = làm V / sẽ làm V.
* [ ] **Vません** = không làm V / sẽ không làm V.
* [ ] **Vました** = đã làm V.
* [ ] **Vませんでした** = đã không làm V.
* [ ] **何時にVますか** = làm V lúc mấy giờ?
* [ ] **何曜日ですか** = thứ mấy?

## 2. Thời gian

* [ ] Biết đọc giờ: 1時、2時、4時、7時、9時.
* [ ] Biết đọc phút cơ bản: 1分、3分、4分、6分、8分、10分.
* [ ] Biết các ngày trong tuần: 月曜日〜日曜日.
* [ ] Biết dùng **午前 / 午後**.
* [ ] Biết dùng **朝、昼、晩、夜**.
* [ ] Biết dùng **毎日、昨日、今日、明日**.

## 3. Động từ thường gặp

* [ ] 起きます = thức dậy.
* [ ] 寝ます = ngủ.
* [ ] 働きます = làm việc.
* [ ] 休みます = nghỉ.
* [ ] 勉強します = học.
* [ ] 終わります = kết thúc.

## 4. Lỗi dễ sai

* [ ] Có thời điểm cụ thể thì dùng **に**:
  7時に起きます。
* [ ] Không dùng **に** với một số từ như 今日、昨日、明日、毎日.
* [ ] Phân biệt hiện tại/tương lai và quá khứ qua đuôi động từ.

---

# F. Bài 5 — Đi/đến/về, phương tiện, người đi cùng

## 1. Mẫu câu chính

* [ ] **Nへ行きます** = đi đến N.
* [ ] **Nへ来ます** = đến N.
* [ ] **Nへ帰ります** = về N.
* [ ] **どこへ行きますか** = đi đâu?
* [ ] **何で行きますか** = đi bằng gì?
* [ ] **Nで行きます** = đi bằng phương tiện N.
* [ ] **だれと行きますか** = đi với ai?
* [ ] **Nと行きます** = đi với N.
* [ ] **いつ行きますか** = khi nào đi?
* [ ] **誕生日は〜月〜日です** = sinh nhật là ngày … tháng …

## 2. Trợ từ chính

* [ ] **へ** = chỉ hướng đến.
* [ ] **に** cũng có thể chỉ điểm đến.
* [ ] **で** = phương tiện.
* [ ] **と** = cùng với ai.
* [ ] **は** = nêu chủ đề.

## 3. Từ cần nắm

* [ ] 行きます = đi.
* [ ] 来ます = đến.
* [ ] 帰ります = về.
* [ ] 学校 = trường học.
* [ ] 会社 = công ty.
* [ ] 駅 = nhà ga.
* [ ] 日本 = Nhật Bản.
* [ ] バス、電車、地下鉄、自転車、歩いて.

## 4. Lỗi dễ sai

* [ ] Phương tiện dùng **で**:
  バスで行きます。
* [ ] Người đi cùng dùng **と**:
  友達と行きます。
* [ ] Đi bộ dùng **歩いて**, không dùng **歩いてで**.
* [ ] Không nhầm **へ** với **で**.

---

# G. Bài 6 — Tân ngữ, hành động, lời mời

## 1. Mẫu câu chính

* [ ] **NをVます** = làm V với N / V N.
* [ ] **Nを食べます** = ăn N.
* [ ] **Nを飲みます** = uống N.
* [ ] **Nを見ます** = xem N.
* [ ] **Nを読みます** = đọc N.
* [ ] **Nを買います** = mua N.
* [ ] **NでVます** = làm V ở địa điểm N.
* [ ] **一緒にVませんか** = cùng làm V nhé?
* [ ] **Vましょう** = hãy cùng làm V / nào.
* [ ] **何をしますか** = làm gì?
* [ ] **どこでVますか** = làm V ở đâu?

## 2. Trợ từ chính

* [ ] **を** = tân ngữ trực tiếp.
* [ ] **で** = nơi diễn ra hành động.
* [ ] **に** trong 一緒に = cùng nhau.
* [ ] **か** = câu hỏi.

## 3. Động từ thường gặp

* [ ] 食べます = ăn.
* [ ] 飲みます = uống.
* [ ] 吸います = hút.
* [ ] 見ます = xem.
* [ ] 聞きます = nghe.
* [ ] 読みます = đọc.
* [ ] 書きます = viết.
* [ ] 買います = mua.
* [ ] 撮ります = chụp.
* [ ] します = làm.
* [ ] 会います = gặp.

## 4. Lỗi dễ sai

* [ ] Tân ngữ dùng **を**:
  ごはんを食べます。
* [ ] Nơi làm hành động dùng **で**:
  レストランで食べます。
* [ ] Gặp ai dùng **に**:
  友達に会います.
* [ ] Không nhầm **へ** là nơi đến với **で** là nơi xảy ra hành động.

---

# H. Bài 7 — Công cụ, cho nhận, lời cảm ơn

## 1. Mẫu câu chính

* [ ] **NでVます** = làm V bằng N / với N.
* [ ] **言語で話します** = nói bằng ngôn ngữ.
* [ ] **Nをあげます** = cho N.
* [ ] **Nをもらいます** = nhận N.
* [ ] **人にNをあげます** = cho ai N.
* [ ] **人に/からNをもらいます** = nhận N từ ai.
* [ ] **もうVました** = đã làm rồi.
* [ ] **まだです** = vẫn chưa.
* [ ] **何でVますか** = làm V bằng gì?
* [ ] **だれにあげますか** = cho ai?
* [ ] **だれにもらいましたか** = nhận từ ai?

## 2. Trợ từ chính

* [ ] **で** = công cụ / phương tiện.
* [ ] **に** = người nhận / người cho trong một số mẫu.
* [ ] **から** = từ ai đó.
* [ ] **を** = vật được cho / nhận.

## 3. Mẫu giao tiếp

* [ ] いらっしゃい。
* [ ] どうぞ。
* [ ] ありがとうございます。
* [ ] もう〜ましたか。
* [ ] はい、もう〜ました。
* [ ] いいえ、まだです。

## 4. Lỗi dễ sai

* [ ] Công cụ dùng **で**:
  はしで食べます。
* [ ] Cho ai dùng **に**:
  友達に本をあげます。
* [ ] Nhận từ ai dùng **に / から**:
  先生に本をもらいました。
* [ ] **もう** đi với việc đã làm.
* [ ] **まだです** là câu trả lời “chưa”.

# I. Bài 8 — Tính từ い và tính từ な

## 1. Mẫu câu chính

* [ ] **NはAdjです** = N thì Adj.
* [ ] **Adjですか** = có Adj không?
* [ ] **とてもAdjです** = rất Adj.
* [ ] **あまりAdjくないです** = không … lắm.
* [ ] **Nはどうですか** = N thế nào?
* [ ] **どんなNですか** = N như thế nào?
* [ ] **Adj Nです** = là N có tính chất Adj.

## 2. Tính từ い

* [ ] おいしい = ngon.
* [ ] 大きい = to.
* [ ] 小さい = nhỏ.
* [ ] 新しい = mới.
* [ ] 古い = cũ.
* [ ] いい = tốt.
* [ ] 悪い = xấu.
* [ ] 暑い = nóng thời tiết.
* [ ] 寒い = lạnh thời tiết.
* [ ] 高い = cao / đắt.
* [ ] 安い = rẻ.
* [ ] 面白い = thú vị.
* [ ] 難しい = khó.
* [ ] 易しい = dễ.
* [ ] 忙しい = bận.

## 3. Tính từ な

* [ ] きれい = đẹp / sạch.
* [ ] 静か = yên tĩnh.
* [ ] にぎやか = náo nhiệt.
* [ ] 有名 = nổi tiếng.
* [ ] 親切 = tốt bụng.
* [ ] 元気 = khỏe.
* [ ] 暇 = rảnh.
* [ ] 便利 = tiện lợi.
* [ ] 素敵 = tuyệt.

## 4. Bổ nghĩa danh từ

* [ ] Tính từ い + N:
  おいしい食べ物.
* [ ] Tính từ な + N:
  きれいな人.
* [ ] Không quên **な** khi tính từ な đứng trước danh từ.

## 5. Lỗi dễ sai

* [ ] きれい là tính từ な, không phải tính từ い.
* [ ] 有名 là tính từ な.
* [ ] Không nói **きれい人**, phải nói **きれいな人**.
* [ ] あまり đi với phủ định:
  あまりおいしくないです.

---

# J. Bài 9 — Sở thích, năng lực, lý do

## 1. Mẫu câu chính

* [ ] **Nが好きです** = thích N.
* [ ] **Nが嫌いです** = ghét / không thích N.
* [ ] **Nが上手です** = giỏi N.
* [ ] **Nが下手です** = kém N.
* [ ] **Nがわかります** = hiểu N.
* [ ] **Nがあります** = có N.
* [ ] **どんなNが好きですか** = thích loại N nào?
* [ ] **よく / だいたい / 少し / あまり / 全然**.
* [ ] **〜から、〜** = vì… nên…
* [ ] **どうして〜か** = tại sao?
* [ ] **〜から** = vì…

## 2. Trợ từ chính

* [ ] Với 好き、嫌い、上手、下手 dùng **が**.
* [ ] Với わかります dùng **が**.
* [ ] Với あります dùng **が**.
* [ ] Lý do dùng **から**.

## 3. Mức độ

* [ ] よくわかります = hiểu rõ.
* [ ] だいたいわかります = hiểu đại khái.
* [ ] 少しわかります = hiểu một chút.
* [ ] あまりわかりません = không hiểu lắm.
* [ ] 全然わかりません = hoàn toàn không hiểu.

## 4. Lỗi dễ sai

* [ ] Không dùng を với 好き:
  音楽が好きです.
* [ ] Không dùng を với わかります:
  日本語がわかります.
* [ ] あまり / 全然 đi với phủ định.
* [ ] から đứng sau lý do.

---

# K. Bài 10 — Tồn tại: あります / います

## 1. Mẫu câu chính

* [ ] **Nがあります** = có N / N tồn tại.
* [ ] **Nがいます** = có người / động vật.
* [ ] **場所にNがあります** = ở nơi có N.
* [ ] **場所にNがいます** = ở nơi có người/động vật.
* [ ] **Nは場所にあります** = N ở nơi.
* [ ] **Nは場所にいます** = người/động vật ở nơi.
* [ ] **何がありますか** = có cái gì?
* [ ] **だれがいますか** = có ai?
* [ ] **Nはどこにありますか** = N ở đâu?
* [ ] **Nはどこにいますか** = người/động vật ở đâu?

## 2. Phân biệt あります / います

* [ ] **あります** dùng cho đồ vật, cây cối, sự kiện.
* [ ] **います** dùng cho người và động vật.
* [ ] 机があります。
* [ ] 猫がいます。
* [ ] 先生がいます。

## 3. Từ chỉ vị trí

* [ ] 上 = trên.
* [ ] 下 = dưới.
* [ ] 前 = trước.
* [ ] 後ろ = sau.
* [ ] 右 = phải.
* [ ] 左 = trái.
* [ ] 中 = trong.
* [ ] 外 = ngoài.
* [ ] 隣 = bên cạnh.
* [ ] 近く = gần.
* [ ] 間 = giữa.

## 4. Lỗi dễ sai

* [ ] Nơi tồn tại dùng **に**:
  部屋に机があります。
* [ ] Nơi xảy ra hành động dùng **で**:
  部屋で勉強します。
* [ ] Đồ vật dùng **あります**.
* [ ] Người/động vật dùng **います**.

---

# L. Bài 11 — Số lượng, thời lượng, tần suất

## 1. Mẫu câu chính

* [ ] **Số lượng + Vます** = làm V số lượng bao nhiêu.
* [ ] **期間 + Vます** = làm V trong bao lâu.
* [ ] **期間に〜回Vます** = trong khoảng thời gian làm mấy lần.
* [ ] **どのくらいVますか** = làm bao lâu?
* [ ] **何人 / 何枚 / 何台 / 何冊 / 何回**.
* [ ] **全部で〜です** = tổng cộng là…
* [ ] **〜だけ** = chỉ…

## 2. Lượng từ cần nắm

* [ ] 人 = người.
* [ ] 枚 = vật mỏng.
* [ ] 台 = máy móc / xe.
* [ ] 冊 = sách.
* [ ] 回 = lần.
* [ ] 個 = vật nhỏ.
* [ ] 本 = vật dài.
* [ ] 杯 = cốc / chén.
* [ ] 匹 = con vật nhỏ.
* [ ] つ = lượng từ chung.

## 3. Thời lượng

* [ ] 1時間 = 1 tiếng.
* [ ] 2時間 = 2 tiếng.
* [ ] 1週間 = 1 tuần.
* [ ] 1か月 = 1 tháng.
* [ ] 1年 = 1 năm.

## 4. Tần suất

* [ ] 1週間に1回 = 1 tuần 1 lần.
* [ ] 1か月に2回 = 1 tháng 2 lần.
* [ ] 毎日 = mỗi ngày.
* [ ] 毎週 = mỗi tuần.
* [ ] 毎月 = mỗi tháng.

## 5. Lỗi dễ sai

* [ ] Số lượng thường đặt trước động từ:
  りんごを3つ買います。
* [ ] Thời lượng không cần trợ từ に trong nhiều trường hợp:
  2時間勉強します。
* [ ] Tần suất dùng **に**:
  1週間に2回勉強します。
* [ ] Phân biệt thời điểm và thời lượng:
  7時に起きます / 2時間勉強します.

# M. Bài 12 — Quá khứ tính từ, so sánh

## 1. Quá khứ của danh từ và tính từ な

* [ ] **Nです → Nでした**
* [ ] **Nじゃありません → Nじゃありませんでした**
* [ ] **なAdjです → なAdjでした**
* [ ] **なAdjじゃありません → なAdjじゃありませんでした**

Ví dụ:

* [ ] 昨日は雨でした。
* [ ] 町は静かでした。
* [ ] ホテルはきれいじゃありませんでした。

## 2. Quá khứ của tính từ い

* [ ] **いAdjです → いAdjかったです**
* [ ] **いAdjくないです → いAdjくなかったです**
* [ ] おいしいです → おいしかったです.
* [ ] 高いです → 高かったです.
* [ ] よくないです → よくなかったです.
* [ ] いいです → よかったです.

## 3. So sánh

* [ ] **N1はN2よりAdjです** = N1 Adj hơn N2.
* [ ] **N1とN2とどちらがAdjですか** = giữa N1 và N2 cái nào Adj hơn?
* [ ] **N1のほうがAdjです** = N1 Adj hơn.
* [ ] **Nの中で何/どこ/だれ/いつが一番Adjですか** = trong N, cái gì/nơi nào/ai/khi nào là nhất?
* [ ] **Nが一番Adjです** = N là nhất.

## 4. Lỗi dễ sai

* [ ] Tính từ い quá khứ bỏ い thêm かった.
* [ ] いいです quá khứ là **よかったです**, không phải いかったです.
* [ ] So sánh hơn dùng **より**.
* [ ] Câu chọn giữa hai thứ dùng **どちら**.
* [ ] Câu nhất dùng **一番**.

# N. Bài 13 — Mong muốn, mục đích đi đâu làm gì

## 1. Mẫu câu chính

* [ ] **Nが欲しいです** = muốn có N.
* [ ] **Vます bỏ ます + たいです** = muốn làm V.
* [ ] **NをVたいです** = muốn làm V với N.
* [ ] **NがVたいです** cũng có thể gặp khi nhấn mạnh đối tượng muốn.
* [ ] **場所へNに行きます** = đi đến nơi để làm N.
* [ ] **場所へVます bỏ ます + に行きます** = đi đến nơi để làm V.
* [ ] **何をしたいですか** = muốn làm gì?
* [ ] **どこへ行きたいですか** = muốn đi đâu?
* [ ] **何が欲しいですか** = muốn cái gì?

## 2. Cách chia たい

* [ ] 食べます → 食べたいです.
* [ ] 飲みます → 飲みたいです.
* [ ] 買います → 買いたいです.
* [ ] 見ます → 見たいです.
* [ ] します → したいです.
* [ ] 来ます → 来たいです.

## 3. Phủ định và quá khứ của たい

Vì **たい** chia như tính từ い:

* [ ] 食べたいです = muốn ăn.
* [ ] 食べたくないです = không muốn ăn.
* [ ] 食べたかったです = đã muốn ăn.
* [ ] 食べたくなかったです = đã không muốn ăn.

## 4. Mục đích đi đâu

* [ ] デパートへ買い物に行きます。
  Đi trung tâm thương mại để mua sắm.
* [ ] 日本へ日本語を勉強しに行きます。
  Đi Nhật để học tiếng Nhật.
* [ ] レストランへ昼ごはんを食べに行きます。
  Đi nhà hàng để ăn trưa.

## 5. Lỗi dễ sai

* [ ] Muốn có đồ vật dùng **欲しいです**.
* [ ] Muốn làm hành động dùng **Vたいです**.
* [ ] Trước **に行きます** chỉ dùng thân ます:
  食べます → 食べに行きます.
* [ ] Không nói **食べたいに行きます**.
* [ ] たい chia như tính từ い.

# Q. Các cặp dễ nhầm bài 1–13

## 1. これ / この

* [ ] これ dùng độc lập.
* [ ] この phải đi với danh từ.
* [ ] これは本です。
* [ ] この本は私のです。

## 2. ここ / これ

* [ ] ここ = nơi này.
* [ ] これ = cái này.

## 3. どこ / どれ / どの

* [ ] どこ = ở đâu.
* [ ] どれ = cái nào.
* [ ] どのN = N nào.

## 4. あります / います

* [ ] Đồ vật dùng あります.
* [ ] Người/động vật dùng います.

## 5. に / で

* [ ] に = nơi tồn tại / thời điểm / điểm đến.
* [ ] で = nơi xảy ra hành động / công cụ / phương tiện.
* [ ] 部屋に机があります。
* [ ] 部屋で勉強します。

## 6. へ / で

* [ ] へ = đi đến đâu.
* [ ] で = làm ở đâu / đi bằng gì.
* [ ] 学校へ行きます。
* [ ] 学校で勉強します。
* [ ] バスで行きます。

## 7. を / が

* [ ] を = tân ngữ hành động.
* [ ] が = sở thích, năng lực, tồn tại, mong muốn.
* [ ] ごはんを食べます。
* [ ] 音楽が好きです。

## 8. もう / まだ

* [ ] もう = đã.
* [ ] まだ = vẫn chưa.
* [ ] もう食べました。
* [ ] まだです。

## 9. 欲しい / たい

* [ ] 欲しい = muốn có đồ vật.
* [ ] たい = muốn làm hành động.
* [ ] 車が欲しいです。
* [ ] 車を買いたいです。

## 10. 時間に / 時間

* [ ] 7時に = lúc 7 giờ.
* [ ] 2時間 = trong 2 tiếng.
* [ ] 7時に起きます。
* [ ] 2時間勉強します。
