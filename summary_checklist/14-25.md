Dưới đây là **checklist kiến thức cần nắm để đi thi Minna bài 14–25**, dựa trên phần chia thể và mẫu ngữ pháp bạn đã đưa. Bạn có thể dùng checklist này để tự soạn bài tập theo từng mục. 

# Checklist tổng hợp Minna bài 14–25

## A. Nền tảng bắt buộc: nhóm động từ

### 1. Nhận diện được 3 nhóm động từ

* [ ] Biết **nhóm 1** là động từ thường đổi âm trước ます theo hàng âm.
* [ ] Biết **nhóm 2** thường bỏ ます rồi thêm đuôi mới.
* [ ] Biết **nhóm 3** gồm します, きます và các động từ dạng **N + します**.
* [ ] Nhận diện được động từ nhóm 1 phổ biến:
  かきます、よみます、いきます、かいます、まちます、はなします
* [ ] Nhận diện được động từ nhóm 2 phổ biến:
  たべます、みます、おきます、ねます、かります、あびます、います
* [ ] Nhận diện được động từ nhóm 3 phổ biến:
  します、きます、勉強します、掃除します、買い物します

### 2. Biết các động từ dễ nhầm nhóm

* [ ] みます là nhóm 2.
* [ ] おきます là nhóm 2.
* [ ] かります là nhóm 2.
* [ ] あびます là nhóm 2.
* [ ] います là nhóm 2.
* [ ] かえります là nhóm 1.
* [ ] はいります là nhóm 1.
* [ ] きります là nhóm 1.
* [ ] いります là nhóm 1 nếu nghĩa là “cần”.

# B. Thể て — bài 14–16

## 1. Cách chia て形

### Nhóm 1

* [ ] い・ち・り + ます → って
  Ví dụ: かいます → かって
* [ ] み・び・に + ます → んで
  Ví dụ: よみます → よんで
* [ ] き + ます → いて
  Ví dụ: かきます → かいて
* [ ] ぎ + ます → いで
  Ví dụ: いそぎます → いそいで
* [ ] し + ます → して
  Ví dụ: はなします → はなして
* [ ] Nhớ ngoại lệ:
  いきます → いって

### Nhóm 2

* [ ] Bỏ ます + て.
* [ ] たべます → たべて
* [ ] みます → みて
* [ ] おきます → おきて

### Nhóm 3

* [ ] します → して
* [ ] きます → きて
* [ ] 勉強します → 勉強して

## 2. Mẫu câu với て形

* [ ] **Vてください** = hãy làm V
  Ví dụ: まってください。
* [ ] **Vてくださいませんか** = bạn có thể làm V giúp tôi không?
* [ ] **Vてもいいです** = được phép làm V
* [ ] **Vてもいいですか** = tôi làm V có được không?
* [ ] **Vてはいけません** = không được làm V
* [ ] **Vています** = đang làm / trạng thái
* [ ] **Vて、Vて、Vます** = làm A rồi B rồi C
* [ ] **Vてから、〜** = sau khi làm V thì…
* [ ] **Vてあげます** = làm V cho người khác
* [ ] **Vてもらいます** = được ai đó làm V cho mình
* [ ] **Vてくれます** = ai đó làm V cho mình
* [ ] **Vてあります** = trạng thái kết quả do ai đó đã làm trước
* [ ] **Vておきます** = làm sẵn / chuẩn bị trước
* [ ] **Vてみます** = thử làm V

## 3. Ý nghĩa cần phân biệt

* [ ] Vています có thể là **đang làm**:
  今、本を読んでいます。
* [ ] Vています có thể là **trạng thái**:
  結婚しています。
  住んでいます。
  知っています。
* [ ] Vて、Vて dùng cho **thứ tự hành động**.
* [ ] Vてから nhấn mạnh **sau khi làm xong V**.
* [ ] Vてはいけません khác với Vないでください:
  một bên là “không được”, một bên là “xin đừng”.

# C. Thể ない — bài 17

## 1. Cách chia ない形

### Nhóm 1

* [ ] Đổi âm trước ます từ hàng **い → あ** + ない.
* [ ] かきます → かかない
* [ ] よみます → よまない
* [ ] はなします → はなさない
* [ ] まちます → またない
* [ ] Nhớ ngoại lệ:
  かいます → かわない
  không phải かあない.

### Nhóm 2

* [ ] Bỏ ます + ない.
* [ ] たべます → たべない
* [ ] みます → みない
* [ ] おきます → おきない

### Nhóm 3

* [ ] します → しない
* [ ] きます → こない
* [ ] 勉強します → 勉強しない

## 2. Mẫu câu với ない形

* [ ] **Vないでください** = xin đừng làm V
* [ ] **Vなければなりません** = phải làm V
* [ ] **Vなければなりませんか** = có phải làm V không?
* [ ] **Vなくてもいいです** = không cần làm V
* [ ] **Vないほうがいいです** = không nên làm V

## 3. Ý nghĩa cần nhớ

* [ ] ない = không làm.
* [ ] ないでください = đừng làm.
* [ ] なければなりません = không làm thì không được → phải làm.
* [ ] なくてもいいです = không làm cũng được → không cần làm.
* [ ] ないほうがいいです = tốt hơn là không nên làm.

# D. 辞書形 / Thể từ điển — bài 18

## 1. Cách chia 辞書形

### Nhóm 1

* [ ] Đổi âm trước ます từ hàng **い → う**.
* [ ] かきます → かく
* [ ] よみます → よむ
* [ ] いきます → いく
* [ ] はなします → はなす
* [ ] かいます → かう
* [ ] まちます → まつ

### Nhóm 2

* [ ] Bỏ ます + る.
* [ ] たべます → たべる
* [ ] みます → みる
* [ ] おきます → おきる
* [ ] ねます → ねる

### Nhóm 3

* [ ] します → する
* [ ] きます → くる
* [ ] 勉強します → 勉強する

## 2. Mẫu câu với 辞書形

* [ ] **Vることができます** = có thể làm V
* [ ] **Nができます** = biết / có thể N
* [ ] **Vるまえに、〜** = trước khi làm V
* [ ] **Nのまえに、〜** = trước N
* [ ] **しゅみは Vることです** = sở thích là làm V
* [ ] **Vる + N** = N mà V / N được bổ nghĩa
* [ ] **Vるとき、〜** = khi làm V
* [ ] **Vるでしょう** = có lẽ sẽ làm V / đúng không tùy ngữ cảnh

## 3. Ý nghĩa cần nhớ

* [ ] 辞書形 là dạng gốc của động từ.
* [ ] Động từ trong từ điển thường kết thúc bằng âm hàng う.
* [ ] Trước **ことができます** phải dùng 辞書形.
* [ ] Trước **まえに** phải dùng 辞書形 nếu là động từ.
* [ ] Khi bổ nghĩa danh từ, động từ đứng trước danh từ ở dạng thường / 辞書形 / た形 / ない形 tùy nghĩa.

# E. Thể た — bài 19

* [ ] Biết た形 chia giống て形.
* [ ] て → た.
* [ ] で → だ.

### Nhóm 1

* [ ] かいます → かって → かった
* [ ] まちます → まって → まった
* [ ] かえります → かえって → かえった
* [ ] よみます → よんで → よんだ
* [ ] あそびます → あそんで → あそんだ
* [ ] しにます → しんで → しんだ

* [ ] いそぎます → いそいで → いそいだ
* [ ] はなします → はなして → はなした
* [ ] Nhớ ngoại lệ:
  いきます → いって → いった

### Nhóm 2

* [ ] たべます → たべた
* [ ] みます → みた
* [ ] おきます → おきた

### Nhóm 3

* [ ] します → した
* [ ] きます → きた
* [ ] 勉強します → 勉強した

## 2. Mẫu câu với た形

* [ ] **Vたことがあります** = đã từng làm V
* [ ] **Vたことがありません** = chưa từng làm V
* [ ] **Vたり、Vたりします** = làm việc này, việc kia

* [ ] Vたことがあります = đã từng có kinh nghiệm.
* [ ] Vて、Vて = hành động theo thứ tự.
* [ ] Vたり、Vたり = liệt kê vài hành động tiêu biểu.
* [ ] もう Vました = đã làm rồi.
* [ ] まだ Vていません = vẫn chưa làm.

# F. 普通形 / Thể thông thường — bài 20–23

## 1. 普通形 của động từ

* [ ] Vます → Vる
* [ ] Vません → Vない
* [ ] Vました → Vた
* [ ] Vませんでした → Vなかった

Ví dụ:

* [ ] たべます → たべる
* [ ] たべません → たべない
* [ ] たべました → たべた
* [ ] たべませんでした → たべなかった

## 2. 普通形 của tính từ い

* [ ] おいしいです → おいしい
* [ ] おいしくないです → おいしくない
* [ ] おいしかったです → おいしかった
* [ ] おいしくなかったです → おいしくなかった

Cần nhớ:

* [ ] Tính từ い ở hiện tại khẳng định chỉ cần bỏ です.
* [ ] Quá khứ khẳng định: い → かった.
* [ ] Phủ định: い → くない.
* [ ] Quá khứ phủ định: い → くなかった.
* [ ] いいです → いい
* [ ] よくないです → よくない
* [ ] よかったです → よかった
* [ ] よくなかったです → よくなかった

## 3. 普通形 của tính từ な

* [ ] しずかです → しずかだ
* [ ] しずかじゃありません → しずかじゃない
* [ ] しずかでした → しずかだった
* [ ] しずかじゃありませんでした → しずかじゃなかった

Cần nhớ:

* [ ] Hiện tại khẳng định thêm **だ**.
* [ ] Phủ định dùng **じゃない**.
* [ ] Quá khứ khẳng định dùng **だった**.
* [ ] Quá khứ phủ định dùng **じゃなかった**.

## 4. 普通形 của danh từ

* [ ] 学生です → 学生だ
* [ ] 学生じゃありません → 学生じゃない
* [ ] 学生でした → 学生だった
* [ ] 学生じゃありませんでした → 学生じゃなかった

## 5. Mẫu câu với 普通形

* [ ] **普通形と思います** = tôi nghĩ rằng…
* [ ] **普通形と言いました** = đã nói rằng…
* [ ] **普通形 + N** = N mà…
* [ ] **普通形 + とき** = khi…
* [ ] **普通形でしょう？** = đúng không?
* [ ] **普通形かもしれません** = có lẽ / có thể…
* [ ] **普通形んです** = giải thích / nhấn mạnh lý do
* [ ] **普通形ので、〜** = vì… nên…
* [ ] **普通形から、〜** = vì… nên…

## 6. Lưu ý với 普通形 + とき

* [ ] Động từ 辞書形 + とき = khi / trước khi làm V.
* [ ] Động từ た形 + とき = khi / sau khi đã làm V.
* [ ] Tính từ い + とき: 忙しいとき
* [ ] Tính từ な + とき: 暇なとき
* [ ] Danh từ + のとき: 子どものとき

# G. たら形 — bài 25

## 1. Cách chia たら形

* [ ] Lấy **た形 + ら**.
* [ ] いきます → いった → いったら
* [ ] たべます → たべた → たべたら
* [ ] よみます → よんだ → よんだら
* [ ] します → した → したら
* [ ] きます → きた → きたら

## 2. Mẫu câu với たら形

* [ ] **Vたら、〜** = nếu V thì…
* [ ] **Vたら、〜** = sau khi V thì…
* [ ] **い形かったら、〜** = nếu tính từ い…
* [ ] **な形だったら、〜** = nếu tính từ な…
* [ ] **Nだったら、〜** = nếu là N…

## 3. Ý nghĩa cần phân biệt

* [ ] たら có thể mang nghĩa **nếu**.
* [ ] たら có thể mang nghĩa **sau khi**.
* [ ] Dựa vào ngữ cảnh để dịch.
* [ ] 雨が降ったら、行きません。= nếu trời mưa thì không đi.
* [ ] 家へ帰ったら、宿題をします。= sau khi về nhà thì làm bài tập.

# I. Các cặp dễ nhầm bắt buộc phải phân biệt

## 1. まで và までに

* [ ] **まで** = kéo dài đến mốc thời gian.
* [ ] **までに** = trước hạn / deadline.
* [ ] 5時まで勉強します。= học đến 5 giờ.
* [ ] 5時までに宿題を出します。= nộp bài trước 5 giờ.

## 2. Vました và Vたことがあります

* [ ] **Vました** = đã làm việc cụ thể.
* [ ] **Vたことがあります** = đã từng có kinh nghiệm.
* [ ] 昨日、すしを食べました。= Hôm qua ăn sushi.
* [ ] すしを食べたことがあります。= Đã từng ăn sushi.

## 3. Vて、Vて và Vたり、Vたり

* [ ] **Vて、Vて** = hành động theo thứ tự.
* [ ] **Vたり、Vたり** = liệt kê vài hoạt động tiêu biểu.
* [ ] 朝ごはんを食べて、学校へ行きます。= Ăn sáng rồi đi học.
* [ ] 日曜日、映画を見たり、買い物したりします。= Chủ nhật xem phim, mua sắm…

## 4. Vてください và Vないでください

* [ ] Vてください = hãy làm.
* [ ] Vないでください = xin đừng làm.

## 5. Vてもいいです và Vてはいけません

* [ ] Vてもいいです = được phép làm.
* [ ] Vてはいけません = không được làm.

## 6. Vなければなりません và Vなくてもいいです

* [ ] Vなければなりません = phải làm.
* [ ] Vなくてもいいです = không cần làm.

## 7. Vるまえに và Vたあとで

* [ ] Vるまえに = trước khi làm.
* [ ] Vたあとで = sau khi làm.

## 8. Vるとき và Vたとき

* [ ] Vるとき = khi / trước lúc làm.
* [ ] Vたとき = khi / sau khi đã làm.

## 9. たら nghĩa “nếu” và “sau khi”

* [ ] Nếu: 雨が降ったら、行きません。
* [ ] Sau khi: 家へ帰ったら、宿題をします。

## 10. Vています: đang làm và trạng thái

* [ ] 今、本を読んでいます。= đang đọc sách.
* [ ] 結婚しています。= đã kết hôn / đang trong trạng thái kết hôn.
* [ ] 住んでいます。= đang sống.
* [ ] 知っています。= biết.

# J. Trợ từ cần nắm để làm bài tốt

## 1. Trợ từ を

* [ ] Dùng với tân ngữ trực tiếp.
* [ ] ごはんを食べます。
* [ ] 本を読みます。
* [ ] 写真を撮ります。

## 2. Trợ từ へ / に

* [ ] Dùng với hướng / điểm đến.
* [ ] 学校へ行きます。
* [ ] 日本に行きます。

## 3. Trợ từ で

* [ ] Nơi diễn ra hành động:
  図書館で勉強します。
* [ ] Phương tiện / công cụ:
  バスで行きます。
  はしで食べます。

## 4. Trợ từ に

* [ ] Thời điểm cụ thể:
  7時に起きます。
* [ ] Gặp ai:
  友達に会います。
* [ ] Nhận từ ai / cho ai trong mẫu cho nhận.

## 5. Trợ từ と

* [ ] Với ai đó:
  友達と行きます。
* [ ] Nội dung trích dẫn:
  雨が降ると思います。

## 6. Trợ từ が

* [ ] Khả năng:
  日本語ができます。
* [ ] Chủ ngữ trong mệnh đề phụ:
  私が買った本
* [ ] Sở thích / năng lực / tính chất:
  料理が好きです。

## 7. Trợ từ は

* [ ] Nêu chủ đề câu.
* [ ] 私は学生です。
* [ ] 日本語は難しいです。

# K. Từ khóa nhận diện mẫu trong bài thi

* [ ] ください → thường cần て形 hoặc ない形 + でください.

* [ ] てはいけません → cần て形.
* [ ] なければなりません → cần ない形 bỏ い thành ければ.

* [ ] ことがあります → cần た形.
* [ ] たり、たりします → cần た形.

# O. Bảng chốt siêu ngắn để dán lên bàn học

| Thể | Cách nhớ           | Mẫu chính              |
| --- | ------------------ | ---------------------- |
| て形  | nối / hãy          | Vてください, Vてもいいです, Vています |
| ない形 | không              | Vないでください, Vなければなりません   |
| 辞書形 | gốc                | Vることができます, Vるまえに       |
| た形  | đã                 | Vたことがあります, VたりVたりします   |
| 普通形 | thân mật / câu dài | 普通形と思います, 普通形 + N      |
| たら形 | nếu / sau khi      | Vたら                    |

Câu chốt để tự học:

**Xác định nhóm → chia thể → ghép mẫu → hiểu nghĩa → tránh lỗi dễ nhầm.**
