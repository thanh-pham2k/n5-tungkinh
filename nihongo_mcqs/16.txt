# 2. V1て、V2て、V3ます — Nối chuỗi hành động

## Công thức

```text
V1て、V2て、V3ます。
```

## Nghĩa

**Làm V1, rồi làm V2, rồi làm V3.**

Dùng để liệt kê các hành động **theo đúng trình tự thời gian**.

Ví dụ:

| Câu                        | Nghĩa                                      |
| -------------------------- | ------------------------------------------ |
| 朝 起きて、歯を磨いて、朝ごはんを食べます。     | Buổi sáng thức dậy, đánh răng rồi ăn sáng. |
| うちへ帰って、シャワーを浴びて、料理しました。    | Về nhà, tắm rồi nấu ăn.                    |
| 9時に起きて、ジョギングして、シャワーを浴びました。 | Thức dậy lúc 9 giờ, chạy bộ rồi tắm.       |

## Lưu ý quan trọng

Động từ cuối câu quyết định **thì của cả câu**.

| Câu       | Nghĩa             |
| --------- | ----------------- |
| 食べて、寝ます。  | Ăn rồi ngủ.       |
| 食べて、寝ました。 | Đã ăn rồi ngủ.    |
| 食べて、寝ません。 | Ăn rồi không ngủ. |

# 3. Vてから、Vます — Sau khi V thì…

## Công thức

```text
Vてから、Vます。
```

## Nghĩa

**Sau khi làm V1 xong thì làm V2.**

Khác với nối hành động thông thường, mẫu này nhấn mạnh **V1 phải xong rồi mới làm V2**.

Ví dụ:

| Câu                    | Nghĩa                                     |
| ---------------------- | ----------------------------------------- |
| レポートを書いてから、ごはんを食べます。   | Viết báo cáo xong rồi ăn cơm.             |
| お客さんに会ってから、社長に電話をかけます。 | Gặp khách hàng xong rồi gọi cho giám đốc. |
| 宿題をしてから、寝ます。           | Làm bài tập xong rồi ngủ.                 |
| ごはんを食べてから、薬を飲んでください。   | Ăn cơm xong rồi hãy uống thuốc.           |
| 勉強してから、遊んでもいいです。       | Học xong rồi đi chơi cũng được.           |

# 4. Nối tính từ い

## Công thức

```text
いAdj bỏ い + くて
```

## Nghĩa

**Vừa Adj1 vừa Adj2.**

Dùng khi các tính chất cùng hướng: cùng tốt hoặc cùng xấu.

Ví dụ biến đổi:

| Gốc | Nối câu |
| --- | ------- |
| 安い  | 安くて     |
| 大きい | 大きくて    |
| 小さい | 小さくて    |
| 白い  | 白くて     |
| いい  | よくて     |

Ví dụ câu:

| Câu                | Nghĩa                              |
| ------------------ | ---------------------------------- |
| このパンは 安くて、おいしいです。  | Bánh mì này rẻ và ngon.            |
| このかばんは 大きくて、重いです。  | Cái cặp này to và nặng.            |
| 東京は 人が多くて、物価が高いです。 | Tokyo đông người và giá cả đắt đỏ. |

# 5. Nối tính từ な

## Công thức

```text
なAdj bỏ な + で
```

## Nghĩa

**Vừa Adj1 vừa Adj2.**

Ví dụ biến đổi:

| Gốc   | Nối câu |
| ----- | ------- |
| きれいな  | きれいで    |
| ハンサムな | ハンサムで   |
| 便利な   | 便利で     |
| 有名な   | 有名で     |
| 親切な   | 親切で     |

Ví dụ câu:

| Câu                   | Nghĩa                             |
| --------------------- | --------------------------------- |
| 桜は きれいで、有名です。         | Hoa anh đào đẹp và nổi tiếng.     |
| タイさんは ハンサムで、日本語が上手です。 | Tài đẹp trai và giỏi tiếng Nhật.  |
| この町は 静かで、便利です。        | Khu phố này yên tĩnh và tiện lợi. |

# 6. Nối tính từ い và な trong cùng câu

Nếu trong câu có cả tính từ **い** và **な**, tính từ nào cần nối thì chia theo đúng loại đó.

Ví dụ:

| Câu               | Nghĩa                                  |
| ----------------- | -------------------------------------- |
| このドレスは きれいで、安いです。 | Cái váy này đẹp và rẻ.                 |
| ベトナム人は 親切で、優しいです。 | Người Việt Nam thân thiện và tốt bụng. |
| 私のうちは 広くて、きれいです。  | Nhà tôi rộng và sạch.                  |

---

# 7. Khi tính chất trái ngược: dùng が

Nếu một ý tốt và một ý xấu, không dùng nối thường. Dùng **が** = nhưng.

```text
Câu 1 が、Câu 2。
```

Ví dụ:

| Câu                     | Nghĩa                              |
| ----------------------- | ---------------------------------- |
| この時計は きれいですが、高いです。      | Đồng hồ này đẹp nhưng đắt.         |
| ナムさんは ハンサムですが、日本語が下手です。 | Nam đẹp trai nhưng kém tiếng Nhật. |
| この料理は おいしいですが、高いです。     | Món này ngon nhưng đắt.            |

# 8. Nối danh từ

## Công thức

```text
N1 で、N2 です。
```

## Nghĩa

**Là N1 và là N2.**

Ví dụ:

| Câu                        | Nghĩa                                               |
| -------------------------- | --------------------------------------------------- |
| マイケルさんは アメリカ人で、さくら大学の先生です。 | Michael là người Mỹ và là giáo viên Đại học Sakura. |
| アンさんは 22歳で、医者です。           | An 22 tuổi và là bác sĩ.                            |
| 私は ベトナム人で、留学生です。           | Tôi là người Việt Nam và là du học sinh.            |
| ゆきさんは ナムさんの友達で、銀行員です。      | Yuki là bạn của Nam và là nhân viên ngân hàng.      |

---

# 9. Nは bộ phậnが Adjです

## Công thức

```text
N は bộ phận が Adj です。
```

## Nghĩa

**N có bộ phận mang tính chất Adj.**

Dùng để miêu tả một phần / đặc điểm của người, vật, địa điểm.

Ví dụ:

| Câu             | Nghĩa                    |
| --------------- | ------------------------ |
| チーさんは 肌が白いです。   | Bạn Chi có da trắng.     |
| ハノイは 物価が高いです。   | Hà Nội có giá cả đắt đỏ. |
| 日本は 食べ物がおいしいです。 | Nhật Bản có đồ ăn ngon.  |
| ハロンは 海がきれいです。   | Hạ Long có biển đẹp.     |
| 彼は 背が高いです。      | Anh ấy cao.              |

---

# 10. Địa điểmを出ます — Ra khỏi địa điểm

## Công thức

```text
Địa điểm を 出ます。
```

## Nghĩa

**Ra khỏi địa điểm.**

Ví dụ:

| Câu       | Nghĩa              |
| --------- | ------------------ |
| うちを 出ます。  | Ra khỏi nhà.       |
| 事務所を 出ます。 | Ra khỏi văn phòng. |
| 喫茶店を 出ます。 | Ra khỏi quán nước. |

## Nghĩa mở rộng

Nếu địa điểm là **trường học / công ty**, mẫu này có thể mang nghĩa:

| Câu          | Nghĩa                              |
| ------------ | ---------------------------------- |
| 高校を 出ました。    | Đã tốt nghiệp cấp 3.               |
| 大学を 出ます。     | Sẽ tốt nghiệp đại học.             |
| 父は 会社を 出ました。 | Bố tôi đã nghỉ việc / rời công ty. |

# 11. Phương tiệnを降ります — Xuống khỏi phương tiện

## Công thức

```text
Phương tiện を 降ります。
```

## Nghĩa

**Xuống khỏi phương tiện.**

Ví dụ:

| Câu         | Nghĩa           |
| ----------- | --------------- |
| バスを 降ります。   | Xuống xe buýt.  |
| 電車を 降ります。   | Xuống tàu điện. |
| タクシーを 降ります。 | Xuống taxi.     |
| 馬を 降ります。    | Xuống ngựa.     |

Ghi nhớ nhanh:

| Hành động       | Trợ từ |
| --------------- | ------ |
| Lên / vào       | に      |
| Xuống / ra khỏi | を      |

---

# 12. どうやって — Bằng cách nào?

## Công thức

```text
どうやって Vますか。
```

## Nghĩa

**Làm V bằng cách nào? / Làm như thế nào?**

Dùng để hỏi **cách làm / quy trình / cách sử dụng máy móc**.

Ví dụ:

| Câu                | Nghĩa                        |
| ------------------ | ---------------------------- |
| この機械は どうやって 使いますか。 | Máy này sử dụng như thế nào? |
| どうやって 漢字を 勉強しますか。  | Học Kanji bằng cách nào?     |
| どうやって 恋人が できますか。   | Làm thế nào để có người yêu? |

Cách trả lời thường dùng chuỗi hành động thể て:

```text
スイッチを入れて、このボタンを押してください。
```

= Bật công tắc rồi hãy bấm nút này.

# Tổng kết cực ngắn

| Mẫu              | Nghĩa                     | Ví dụ             |
| ---------------- | ------------------------- | ----------------- |
| `Vて、Vて、Vます`      | làm V1, rồi V2, rồi V3    | 起きて、歯を磨いて、学校へ行きます |
| `Vてから、Vます`       | sau khi V xong thì…       | 食べてから、薬を飲みます      |
| `いAdj bỏ い + くて` | tính từ い nối câu         | 安くて、おいしいです        |
| `なAdj bỏ な + で`  | tính từ な nối câu         | きれいで、有名です         |
| `Nで、Nです`         | là N và là N              | ベトナム人で、会社員です      |
| `Nは 部分が Adjです`   | N có bộ phận/đặc điểm Adj | 日本は 食べ物がおいしいです    |
| `場所を 出ます`        | ra khỏi địa điểm          | 事務所を 出ます          |
| `乗り物を 降ります`      | xuống phương tiện         | 電車を 降ります          |
| `乗り物に 乗ります`      | lên phương tiện           | バスに 乗ります          |
| `どうやって`          | bằng cách nào             | どうやって 使いますか       |
