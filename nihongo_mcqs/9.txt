# 2. Nが好きです / 嫌いです

## Công thức

```text
N が 好きです。
N が 嫌いです。
```

## Nghĩa

**Thích N / ghét N**

Lưu ý: **好き** và **嫌い** là **tính từ な**, không phải động từ.

| Câu             | Nghĩa                   |
| --------------- | ----------------------- |
| ダンスが 好きです。      | Tôi thích nhảy.         |
| たまごと やさいが 好きです。 | Tôi thích trứng và rau. |
| あめが 嫌いです。       | Tôi ghét mưa.           |
| からい 食べ物が 嫌いです。  | Tôi ghét đồ ăn cay.     |

---

## Nが好きじゃありません

Mẫu này thường dùng hơn **嫌いです** vì nhẹ hơn.

```text
N が 好きじゃありません。
```

= Không thích N.

| Câu              | Nghĩa                     |
| ---------------- | ------------------------- |
| スポーツが 好きじゃありません。 | Tôi không thích thể thao. |
| 勉強が 好きじゃありません。   | Tôi không thích học.      |
| くすりが 好きじゃありません。  | Tôi không thích thuốc.    |

# 3. Câu hỏi với 好き / 嫌い

## 3.1. Câu hỏi xác nhận

```text
N が 好きですか。
N が 嫌いですか。
```

= Có thích / ghét N không?

| Câu hỏi      | Nghĩa                              |
| ------------ | ---------------------------------- |
| 野球が 好きですか。   | Bạn có thích bóng chày không?      |
| 中国料理が 好きですか。 | Bạn có thích món Trung Quốc không? |
| 日本語が 好きですか。  | Bạn có thích tiếng Nhật không?     |

## Trả lời

| Trả lời            | Nghĩa                       |
| ------------------ | --------------------------- |
| はい、好きです。           | Vâng, tôi thích.            |
| はい、とても 好きです。       | Vâng, tôi rất thích.        |
| いいえ、好きじゃありません。     | Không, tôi không thích.     |
| いいえ、あまり 好きじゃありません。 | Không, tôi không thích lắm. |

---

## 3.2. なにが好きですか

```text
なに が 好きですか。
```

= Bạn thích gì?

Ví dụ:

```text
なにが 好きですか。
お金が 好きです。
```

= Bạn thích gì?
= Tôi thích tiền.

---

## 3.3. どんなNが好きですか

```text
どんな N が 好きですか。
```

= Bạn thích loại N như thế nào?

| Câu hỏi          | Nghĩa                               |
| ---------------- | ----------------------------------- |
| どんな 料理が 好きですか。   | Bạn thích món ăn như thế nào?       |
| どんな 人が 好きですか。    | Bạn thích người như thế nào?        |
| どんな スポーツが 好きですか。 | Bạn thích môn thể thao như thế nào? |
| どんな 音楽が 好きですか。   | Bạn thích loại nhạc nào?            |

# 4. Nが上手です / 下手です

## Công thức

```text
N が 上手です。
N が 下手です。
```

## Nghĩa

**Giỏi N / kém N**

| Câu        | Nghĩa            |
| ---------- | ---------------- |
| テニスが 上手です。 | Giỏi tennis.     |
| 日本語が 上手です。 | Giỏi tiếng Nhật. |
| 歌が 上手です。   | Hát hay.         |
| ダンスが 上手です。 | Nhảy giỏi.       |
| 日本語が 下手です。 | Kém tiếng Nhật.  |
| 料理が 下手です。  | Nấu ăn kém.      |

Lưu ý: **上手** thường dùng để khen người khác, ít dùng để tự khen mình.

---

## Nが上手じゃありません

```text
N が 上手じゃありません。
```

= Không giỏi N.

Cách này nhẹ hơn nói **下手です**.

| Câu              | Nghĩa                    |
| ---------------- | ------------------------ |
| スポーツが 上手じゃありません。 | Tôi không giỏi thể thao. |
| 勉強が 上手じゃありません。   | Tôi học không giỏi.      |
| 料理が 上手じゃありません。   | Tôi nấu ăn không giỏi.   |

# 5. Nがあります

## Công thức

```text
N が あります。
N が ありません。
```

## Nghĩa

**Có N / không có N**

Dùng với **đồ vật, thời gian, việc bận, cuộc hẹn**.

| Câu        | Nghĩa               |
| ---------- | ------------------- |
| 車が あります。   | Có ô tô.            |
| 用事が あります。  | Có việc bận.        |
| 約束が あります。  | Có cuộc hẹn.        |
| 時間が ありません。 | Không có thời gian. |
| お金が ありません。 | Không có tiền.      |

Lưu ý: **あります** dùng cho vật/sự việc. Với người/động vật sẽ học mẫu khác là **います**.

---

# 6. Phó từ đi với あります

| Phó từ | Mẫu       | Nghĩa              |
| ------ | --------- | ------------------ |
| たくさん   | たくさん あります | Có nhiều           |
| 少し     | 少し あります   | Có một ít          |
| あまり    | あまり ありません | Không có nhiều lắm |
| 全然     | 全然 ありません  | Hoàn toàn không có |

Ví dụ:

| Câu               | Nghĩa                         |
| ----------------- | ----------------------------- |
| プレゼントが たくさん あります。 | Có nhiều quà.                 |
| 細かいお金が 少し あります。   | Có một ít tiền lẻ.            |
| 服が あまり ありません。     | Không có nhiều quần áo lắm.   |
| 時間が 全然 ありません。     | Hoàn toàn không có thời gian. |

---

# 7. Nがわかります

## Công thức

```text
N が わかります。
N が わかりません。
```

## Nghĩa

**Hiểu / biết về N**

Dùng với **ngôn ngữ, lĩnh vực, kiến thức, bài học**.

| Câu           | Nghĩa                    |
| ------------- | ------------------------ |
| 漢字が わかります。    | Hiểu chữ Hán.            |
| 日本語が わかります。   | Hiểu tiếng Nhật.         |
| 英語が わかります。    | Hiểu tiếng Anh.          |
| この問題が わかりません。 | Không hiểu bài này.      |
| ひらがなが わかりません。 | Không hiểu chữ Hiragana. |

---

# 8. Phó từ đi với わかります

| Phó từ | Mẫu        | Nghĩa                |
| ------ | ---------- | -------------------- |
| よく     | よく わかります   | Hiểu rõ              |
| だいたい   | だいたい わかります | Hiểu đại khái        |
| 少し     | 少し わかります   | Hiểu một chút        |
| あまり    | あまり わかりません | Không hiểu lắm       |
| 全然     | 全然 わかりません  | Hoàn toàn không hiểu |

Ví dụ:

| Câu               | Nghĩa                                |
| ----------------- | ------------------------------------ |
| 英語が よく わかります。     | Tôi hiểu rõ tiếng Anh.               |
| 漢字が だいたい わかります。   | Tôi hiểu đại khái chữ Hán.           |
| 英語が 少し わかります。     | Tôi hiểu một chút tiếng Anh.         |
| フランス語が 全然 わかりません。 | Tôi hoàn toàn không hiểu tiếng Pháp. |

# 9. から — vì / bởi vì

## Công thức

```text
Câu 1 から、Câu 2。
```

## Nghĩa

**Vì câu 1 nên câu 2**

| Câu                     | Nghĩa                                   |
| ----------------------- | --------------------------------------- |
| 時間が ありませんから、ごはんを 食べません。 | Vì không có thời gian nên không ăn cơm. |
| お金が ありませんから、どこも 行きません。  | Vì không có tiền nên không đi đâu cả.   |
| この シャツは 高いですから、買いません。   | Vì cái áo này đắt nên không mua.        |
| 忙しいですから、ふるさとへ 帰りません。    | Vì bận nên không về quê.                |

Có thể nói **kết quả trước, lý do sau**:

```text
ごはんを 食べませんでした。時間が ありませんでしたから。
```

= Tôi đã không ăn cơm. Vì không có thời gian.

# 10. どうして — tại sao

## Công thức

```text
どうして + câu hỏi?
```

## Nghĩa

**Tại sao? / Vì sao?**

Câu trả lời thường kết thúc bằng **から**.

| Câu hỏi            | Trả lời         |
| ------------------ | --------------- |
| どうして 晩ごはんを 食べませんか。 | ダイエットですから。      |
| どうして 日本語を 勉強しますか。  | 日本語は おもしろいですから。 |
| どうして 薬を 飲みませんか。    | 薬が 好きじゃありませんから。 |
| どうして 遅れましたか。       | けさ 遅く 起きましたから。  |

# Tổng kết nhanh cần nhớ

| Mẫu            | Nghĩa            | Ví dụ          |
| -------------- | ---------------- | -------------- |
| N が好きです        | Thích N          | 日本語が好きです。      |
| N が嫌いです        | Ghét N           | あめが嫌いです。       |
| N が上手です        | Giỏi N           | テニスが上手です。      |
| N が下手です        | Kém N            | 料理が下手です。       |
| N があります        | Có N             | 時間があります。       |
| N がありません       | Không có N       | お金がありません。      |
| N がわかります       | Hiểu N           | 漢字がわかります。      |
| あまり + phủ định | Không… lắm       | あまりわかりません。     |
| 全然 + phủ định  | Hoàn toàn không… | 全然ありません。       |
| Câu から、Câu     | Vì… nên…         | 忙しいですから、行きません。 |

**Điểm quan trọng nhất:**
Với **好き・嫌い・上手・下手・あります・わかります**, thường dùng trợ từ **が**.
