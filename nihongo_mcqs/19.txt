# 2. Thể た là gì?

## Ý nghĩa

**Thể た** là dạng quá khứ thông thường của động từ.

Nó tương đương với **Vました**.

| Lịch sự | Thông thường |
| ------- | ------------ |
| 食べました   | 食べた          |
| 行きました   | 行った          |
| 読みました   | 読んだ          |
| しました    | した           |
| 来ました    | 来た           |

Nghĩa chung: **đã làm V**.

Ví dụ:

| Câu      | Nghĩa        |
| -------- | ------------ |
| ごはんを食べた。 | Đã ăn cơm.   |
| 日本へ行った。  | Đã đi Nhật.  |
| 本を読んだ。   | Đã đọc sách. |

---

# 3. Cách chia thể た

## 3.1. Nhóm 1

Cách chia giống **thể て**, chỉ đổi:

```text
て → た
で → だ
```

| Vます  | Vて  | Vた  | Nghĩa         |
| ---- | --- | --- | ------------- |
| 書きます | 書いて | 書いた | đã viết       |
| 聞きます | 聞いて | 聞いた | đã nghe       |
| 泳ぎます | 泳いで | 泳いだ | đã bơi        |
| 読みます | 読んで | 読んだ | đã đọc        |
| 遊びます | 遊んで | 遊んだ | đã chơi       |
| 死にます | 死んで | 死んだ | đã chết       |
| 買います | 買って | 買った | đã mua        |
| 待ちます | 待って | 待った | đã đợi        |
| 走ります | 走って | 走った | đã chạy       |
| 話します | 話して | 話した | đã nói chuyện |

Lưu ý đặc biệt:

| Vます  | Vた  |
| ---- | --- |
| 行きます | 行った |

---

## 3.2. Nhóm 2

Quy tắc:

```text
Bỏ ます + た
```

| Vます  | Vた  | Nghĩa                   |
| ---- | --- | ----------------------- |
| 食べます | 食べた | đã ăn                   |
| 見ます  | 見た  | đã xem                  |
| 寝ます  | 寝た  | đã ngủ                  |
| 起きます | 起きた | đã thức dậy             |
| 借ります | 借りた | đã mượn                 |
| 降ります | 降りた | đã xuống                |
| 浴びます | 浴びた | đã tắm                  |
| できます | できた | đã có thể / đã làm được |

## 3.3. Nhóm 3

| Vます    | Vた     | Nghĩa       |
| ------ | ------ | ----------- |
| します    | した     | đã làm      |
| 勉強します  | 勉強した   | đã học      |
| 散歩します  | 散歩した   | đã đi dạo   |

| 持って来ます | 持って来た  | đã mang đến |

# 4. Vたことがあります — Đã từng làm V

## Công thức

```text
Vた + ことがあります。
```

## Nghĩa

**Đã từng làm V.**

Dùng để nói về **kinh nghiệm / trải nghiệm trong quá khứ**.

Ví dụ:

| Câu               | Nghĩa                            |
| ----------------- | -------------------------------- |
| オバマさんに会ったことがあります。 | Tôi đã từng gặp ông Obama.       |
| 富士山に登ったことがあります。   | Tôi đã từng leo núi Phú Sĩ.      |
| 一人で雨を見たことがあります。   | Tôi đã từng ngắm mưa một mình.   |
| あの病院に入ったことがあります。  | Tôi đã từng vào bệnh viện kia.   |
| アイドルと歌ったことがあります。  | Tôi đã từng hát cùng thần tượng. |
| タイへ行ったことがあります。    | Tôi đã từng đi Thái Lan.         |

Ghi nhớ:

```text
食べました = đã ăn
食べたことがあります = đã từng ăn / có trải nghiệm ăn
```

# 5. Vたことがありません — Chưa từng làm V

## Công thức

```text
Vた + ことがありません。
```

## Nghĩa

**Chưa từng làm V.**

Dùng khi bản thân **chưa có kinh nghiệm / chưa có cơ hội làm việc đó**.

Ví dụ:

| Câu                   | Nghĩa                                   |
| --------------------- | --------------------------------------- |
| 日本へ行ったことがありません。       | Tôi chưa từng đi Nhật.                  |
| 恋人と買い物に行ったことがありません。   | Tôi chưa từng đi mua sắm với người yêu. |
| だれともデートしたことがありません。    | Tôi chưa từng hẹn hò với ai.            |
| 車を運転したことがありません。       | Tôi chưa từng lái ô tô.                 |
| 花見をしたことがありません。        | Tôi chưa từng ngắm hoa anh đào.         |
| 両親にありがとうと言ったことがありません。 | Tôi chưa từng nói cảm ơn bố mẹ.         |
| だれにも嘘を言ったことがありません。    | Tôi chưa từng nói dối ai.               |

# 6. Vたことがありますか — Đã từng làm V chưa?

## Công thức

```text
Vた + ことがありますか。
```

## Nghĩa

**Bạn đã từng làm V chưa?**

Cách trả lời:

| Trả lời    | Nghĩa         |
| ---------- | ------------- |
| はい、あります。   | Rồi, đã từng. |
| いいえ、ありません。 | Chưa từng.    |

Ví dụ:

| Câu hỏi           | Nghĩa                                    |
| ----------------- | ---------------------------------------- |
| 日本へ行ったことがありますか。   | Bạn đã từng đi Nhật chưa?                |
| すしを食べたことがありますか。   | Bạn đã từng ăn sushi chưa?               |
| 富士山に登ったことがありますか。  | Bạn đã từng leo núi Phú Sĩ chưa?         |
| 日本で月を見たことがありますか。  | Bạn đã từng ngắm trăng ở Nhật chưa?      |
| バイクで旅行したことがありますか。 | Bạn đã từng đi du lịch bằng xe máy chưa? |

# 7. Vたり、Vたりします — Liệt kê hành động tiêu biểu

## Công thức

```text
V1たり、V2たりします。
```

## Nghĩa

**Làm V1, làm V2,... / lúc thì V1, lúc thì V2.**

Dùng để liệt kê **một vài hành động tiêu biểu**, không cần đúng trình tự thời gian.

Ví dụ:

| Câu                                 | Nghĩa                                            |
| ----------------------------------- | ------------------------------------------------ |
| 映画を見たり、牛乳を買ったりします。                  | Tôi xem phim, mua sữa,...                        |
| 寝る前に、音楽を聞いたり、本を読んだりします。             | Trước khi ngủ, tôi nghe nhạc, đọc sách,...       |
| 今日は掃除したり、洗濯したりしなければなりません。           | Hôm nay tôi phải dọn dẹp, giặt đồ,...            |
| 休みの日は買い物したり、映画を見たりしたいです。            | Ngày nghỉ tôi muốn mua sắm, xem phim,...         |
| ダイエットのために、りんごを食べたり、水を飲んだり、歩いたりしました。 | Để giảm cân, tôi đã ăn táo, uống nước, đi bộ,... |

So sánh nhanh:

| Mẫu        | Ý nghĩa                              |
| ---------- | ------------------------------------ |
| Vて、Vて、Vます  | Làm theo trình tự thời gian          |
| Vたり、Vたりします | Liệt kê ví dụ, không cần theo thứ tự |

# 8. なにをしますか với Vたり

## Câu hỏi

```text
なにをしますか。
```

= Bạn làm gì?

Ví dụ:

| Câu hỏi                | Trả lời                   |
| ---------------------- | ------------------------- |
| 日曜日、なにをしますか。           | 食べたり、寝たりします。              |
| お正月、なにをしたいですか。         | 家族とごはんを食べたり、遊んだりしたいです。    |
| タイでなにをしましたか。           | 写真を撮ったり、タイ料理を食べたりしました。    |
| 結婚する前に、なにをしなければなりませんか。 | 家を買ったり、写真を撮ったりしなければなりません。 |

# 9. ～く / ～に なります — Trở nên / trở thành

## 9.1. Tính từ い + なります

### Công thức

```text
いAdj bỏ い + くなります。
```

## Nghĩa

**Trở nên Adj.**

| Tính từ | Mẫu     | Nghĩa             |
| ------- | ------- | ----------------- |
| 寒い      | 寒くなります  | trở nên lạnh      |
| 暑い      | 暑くなります  | trở nên nóng      |
| 高い      | 高くなります  | trở nên đắt / cao |
| 安い      | 安くなります  | trở nên rẻ        |
| 忙しい     | 忙しくなります | trở nên bận       |

Ví dụ:

| Câu           | Nghĩa                          |
| ------------- | ------------------------------ |
| 天気が寒くなりました。   | Thời tiết đã trở nên lạnh.     |
| 物価が高くなりました。   | Giá cả đã trở nên đắt đỏ.      |
| 日本語が難しくなりました。 | Tiếng Nhật đã trở nên khó hơn. |

---

## 9.2. Tính từ な / danh từ + なります

### Công thức

```text
なAdj bỏ な + になります。
N + になります。
```

## Nghĩa

**Trở nên Adj / trở thành N.**

Ví dụ:

| Câu          | Nghĩa                          |
| ------------ | ------------------------------ |
| 元気になります。     | Trở nên khỏe.                  |
| きれいになります。    | Trở nên sạch / đẹp.            |
| 日本語が上手になります。 | Trở nên giỏi tiếng Nhật.       |
| 先生になります。     | Trở thành giáo viên.           |
| 8歳になります。     | Trở thành 8 tuổi / lên 8 tuổi. |
| 社長になります。     | Trở thành giám đốc.            |

---

## 9.3. Ví dụ tổng hợp

| Câu                   | Nghĩa                                           |
| --------------------- | ----------------------------------------------- |
| 毎日運動しますから、元気になります。    | Vì vận động mỗi ngày nên trở nên khỏe.          |
| 卒業してから、先生になります。       | Sau khi tốt nghiệp, tôi sẽ trở thành giáo viên. |
| 掃除してから、部屋がきれいになりました。  | Sau khi dọn dẹp, phòng đã trở nên sạch.         |
| 彼女に会ってから、彼女が好きになりました。 | Sau khi gặp cô ấy, tôi đã thích cô ấy.          |
| 留学してから、日本語が上手になりました。  | Sau khi đi du học, tiếng Nhật đã giỏi hơn.      |

# Tổng kết cực ngắn

| Mẫu                 | Nghĩa                  | Ví dụ            |
| ------------------- | ---------------------- | ---------------- |
| `Vた`                | đã làm V               | 食べた              |
| `Vたことがあります`         | đã từng làm V          | 日本へ行ったことがあります    |
| `Vたことがありません`        | chưa từng làm V        | 日本へ行ったことがありません   |
| `Vたことがありますか`        | đã từng làm V chưa?    | すしを食べたことがありますか   |
| `はい、あります`           | rồi, đã từng           | はい、あります          |
| `いいえ、ありません`         | chưa từng              | いいえ、ありません        |
| `Vたり、Vたりします`        | làm này, làm kia,...   | 映画を見たり、買い物したりします |
| `いAdj bỏ い + くなります` | trở nên Adj            | 寒くなります           |
| `なAdj + になります`      | trở nên Adj            | 元気になります          |
| `N + になります`         | trở thành N            | 先生になります          |
| `〜なりました`            | đã trở nên / trở thành | 上手になりました         |
