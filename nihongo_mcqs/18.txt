# 2. Thể từ điển / thể nguyên dạng Vる

## Ý nghĩa

**Thể từ điển** là dạng gốc của động từ, dùng để:

| Công dụng             | Ví dụ      |
| --------------------- | ---------- |
| Tra từ điển           | 食べます → 食べる |
| Nói thân mật          | ごはんを食べる    |
| Tạo mẫu ngữ pháp khác | 食べることができます |

Ví dụ:

| Thể ます | Thể từ điển | Nghĩa |
| ------ | ----------- | ----- |
| 食べます   | 食べる         | ăn    |
| 飲みます   | 飲む          | uống  |
| 書きます   | 書く          | viết  |
| 読みます   | 読む          | đọc   |
| 来ます    | 来る          | đến   |
| します    | する          | làm   |

---

# 3. Cách chia thể từ điển

## 3.1. Nhóm 1

Quy tắc:

```text
Âm trước ます thuộc cột い → chuyển sang cột う
```

| Vます  | Vる | Nghĩa    |
| ---- | -- | -------- |
| 書きます | 書く | viết     |
| 読みます | 読む | đọc      |
| 取ります | 取る | lấy      |
| 行きます | 行く | đi       |
| 吸います | 吸う | hút      |
| 遊びます | 遊ぶ | chơi     |
| 待ちます | 待つ | đợi      |
| 貸します | 貸す | cho mượn |
| 思います | 思う | nghĩ     |
| 探します | 探す | tìm kiếm |

---

## 3.2. Nhóm 2

Quy tắc:

```text
Bỏ ます + る
```

| Vます  | Vる  | Nghĩa    |
| ---- | --- | -------- |
| 食べます | 食べる | ăn       |
| 見ます  | 見る  | xem      |
| 寝ます  | 寝る  | ngủ      |
| 起きます | 起きる | thức dậy |
| 借ります | 借りる | mượn     |
| 降ります | 降りる | xuống    |
| 浴びます | 浴びる | tắm      |
| できます | できる | có thể   |

---

## 3.3. Nhóm 3

| Vます   | Vる     | Nghĩa   |
| ----- | ------ | ------- |
| します   | する     | làm     |
| 勉強します | 勉強する   | học     |
| 散歩します | 散歩する   | đi dạo  |
| 結婚します | 結婚する   | kết hôn |
| 来ます   | 来る（くる） | đến     |

# 4. Nができます — Có thể làm N

## Công thức

```text
N が できます。
```

## Nghĩa

**Có thể làm N.**

Danh từ thường là:

| Loại N                | Ví dụ             |
| --------------------- | ----------------- |
| Hành động / hoạt động | 運転, 買い物, スキー      |
| Kỹ năng               | 日本語, 英語, ピアノ, ギター |

Ví dụ:

| Câu             | Nghĩa                      |
| --------------- | -------------------------- |
| ランさんは 日本語ができます。 | Lan có thể nói tiếng Nhật. |
| 運転ができます。        | Tôi có thể lái xe.         |
| ナムさんは ギターができます。 | Nam có thể chơi guitar.    |
| 料理ができます。        | Tôi có thể nấu ăn.         |
| スキーができます。       | Tôi có thể trượt tuyết.    |

---

## Phủ định / nghi vấn

| Mẫu     | Nghĩa               |
| ------- | ------------------- |
| Nができます  | Có thể làm N        |
| Nができません | Không thể làm N     |
| Nができますか | Có thể làm N không? |

Ví dụ:

| Câu        | Nghĩa                    |
| ---------- | ------------------------ |
| ギターができません。 | Không thể chơi guitar.   |
| 料理ができますか。  | Bạn có thể nấu ăn không? |
| はい、できます。   | Vâng, có thể.            |
| いいえ、できません。 | Không, không thể.        |

# 5. Vることができます — Có thể làm V

## Công thức

```text
Vる + ことができます。
```

## Nghĩa

**Có thể làm V.**

Lưu ý: phải có **こと**, vì **Vること** biến động từ thành cụm danh từ.

Ví dụ:

| Câu                 | Nghĩa                     |
| ------------------- | ------------------------- |
| タイさんは 泳ぐことができます。    | Tài có thể bơi.           |
| アンさんは 漢字を読むことができます。 | An có thể đọc Kanji.      |
| カードで払うことができます。      | Có thể trả bằng thẻ.      |
| 日本料理を作ることができます。     | Có thể nấu món Nhật.      |
| 50メートルぐらい泳ぐことができます。 | Có thể bơi khoảng 50 mét. |

---

## Phủ định / nghi vấn

| Mẫu        | Nghĩa               |
| ---------- | ------------------- |
| Vることができます  | Có thể làm V        |
| Vることができません | Không thể làm V     |
| Vることができますか | Có thể làm V không? |

Ví dụ:

| Câu              | Nghĩa                          |
| ---------------- | ------------------------------ |
| 泳ぐことができません。      | Không thể bơi.                 |
| 日本料理を作ることができますか。 | Bạn có thể nấu món Nhật không? |
| いいえ、できません。       | Không, không thể.              |

# 6. 趣味はNです / 趣味はVることです

## 6.1. 趣味はNです

```text
趣味は N です。
```

= Sở thích của tôi là N.

Ví dụ:

| Câu         | Nghĩa                        |
| ----------- | ---------------------------- |
| 趣味は サッカーです。 | Sở thích của tôi là bóng đá. |
| 趣味は 料理です。   | Sở thích của tôi là nấu ăn.  |
| 趣味は 旅行です。   | Sở thích của tôi là du lịch. |
| 趣味は 買い物です。  | Sở thích của tôi là mua sắm. |

---

## 6.2. 趣味はVることです

```text
趣味は Vることです。
```

= Sở thích của tôi là làm V.

Ví dụ:

| Câu            | Nghĩa                          |
| -------------- | ------------------------------ |
| 趣味は 絵を描くことです。  | Sở thích của tôi là vẽ tranh.  |
| 趣味は 写真を撮ることです。 | Sở thích của tôi là chụp ảnh.  |
| 趣味は 音楽を聞くことです。 | Sở thích của tôi là nghe nhạc. |
| 趣味は 本を読むことです。  | Sở thích của tôi là đọc sách.  |
| 趣味は 映画を見ることです。 | Sở thích của tôi là xem phim.  |

---

## 6.3. Hỏi sở thích

```text
趣味は なんですか。
```

= Sở thích của bạn là gì?

Ví dụ trả lời:

| Câu            | Nghĩa                                      |
| -------------- | ------------------------------------------ |
| 趣味は 音楽を聞くことです。 | Sở thích của tôi là nghe nhạc.             |
| 趣味は 旅行や買い物です。  | Sở thích của tôi là du lịch, mua sắm,...   |
| 趣味は 友達と話すことです。 | Sở thích của tôi là nói chuyện với bạn bè. |

# 7. Vる前に、V2 — Trước khi làm V

## Công thức

```text
Vる + 前に、V2
```

## Nghĩa

**Trước khi làm V1 thì làm V2.**

Lưu ý:

| Điểm cần nhớ          | Giải thích                     |
| --------------------- | ------------------------------ |
| V1 luôn ở thể từ điển | Không chia quá khứ             |
| Thì của câu nằm ở V2  | V2 quyết định hiện tại/quá khứ |

Ví dụ:

| Câu                    | Nghĩa                                      |
| ---------------------- | ------------------------------------------ |
| 日本へ来る前に、日本語を勉強しました。    | Trước khi đến Nhật, tôi đã học tiếng Nhật. |
| 寝る前に、本を読みます。           | Trước khi ngủ, tôi đọc sách.               |
| キスする前に、歯を磨かなければなりません。  | Trước khi hôn, phải đánh răng.             |
| 薬を飲む前に、ごはんを食べます。       | Trước khi uống thuốc, ăn cơm.              |
| 会社へ行く前に、朝ごはんを食べませんでした。 | Trước khi đi làm, tôi đã không ăn sáng.    |

---

# 8. Nの前に、V — Trước N thì V

## Công thức

```text
N の 前に、V
```

## Nghĩa

**Trước N thì làm V.**

N thường là danh từ chỉ hành động / sự kiện.

Ví dụ:

| Câu                  | Nghĩa                                     |
| -------------------- | ----------------------------------------- |
| 食事の前に、手を洗わなければなりません。 | Trước bữa ăn, phải rửa tay.               |
| 会議の前に、資料を準備しました。     | Trước cuộc họp, tôi đã chuẩn bị tài liệu. |
| 買い物の前に、お金を下ろします。     | Trước khi mua sắm, tôi rút tiền.          |
| 旅行の前に、何をしますか。        | Trước khi đi du lịch, bạn làm gì?         |
| パーティーの前に、歌いましょう。     | Trước bữa tiệc, cùng hát nhé.             |

---

# 9. Khoảng thời gian + 前に — Cách đây...

## Công thức

```text
Khoảng thời gian + 前に、V
```

## Nghĩa

**Cách đây... đã làm V.**

Không thêm **の** sau khoảng thời gian.

Ví dụ:

| Câu               | Nghĩa                               |
| ----------------- | ----------------------------------- |
| 2時間前に 帰りました。      | Tôi đã về cách đây 2 tiếng.         |
| 2年前に 日本へ来ました。     | Tôi đã đến Nhật cách đây 2 năm.     |
| 3年前に 結婚しました。      | Tôi đã kết hôn cách đây 3 năm.      |
| 1週間前に ベトナムへ帰りました。 | Tôi đã về Việt Nam cách đây 1 tuần. |

---

# 10. なかなかVません — Mãi mà không V

## Công thức

```text
なかなか Vません。
```

## Nghĩa

**Mãi mà không V / khó có thể V.**

Động từ phía sau thường ở dạng phủ định.

Ví dụ:

| Câu                   | Nghĩa                                |
| --------------------- | ------------------------------------ |
| 仕事が多いですから、なかなか終わりません。 | Vì nhiều việc nên mãi mà không xong. |
| 漢字をなかなか覚えません。         | Mãi mà không nhớ Kanji.              |
| バスがなかなか来ません。          | Xe buýt mãi không đến.               |
| 会議がなかなか終わりません。        | Cuộc họp mãi không kết thúc.         |
| みんなの名前をなかなか覚えません。     | Mãi mà không nhớ tên mọi người.      |

Có thể dùng với **Vることができません** để nhấn mạnh “khó có thể làm V”:

| Câu                    | Nghĩa                                  |
| ---------------------- | -------------------------------------- |
| ベトナムで雪を見ることがなかなかできません。 | Ở Việt Nam khó có thể nhìn thấy tuyết. |
| 家を買うことがなかなかできません。      | Khó có thể mua nhà.                    |

# 11. ぜひ — Nhất định / rất mong

## Công thức thường gặp

```text
ぜひ Vたいです。
ぜひ Vてください。
```

## Nghĩa

**Nhất định / rất muốn / rất mong.**

Dùng để nhấn mạnh mong muốn mạnh.

Ví dụ:

| Câu             | Nghĩa                           |
| --------------- | ------------------------------- |
| ぜひ 日本へ行きたいです。   | Tôi rất muốn đi Nhật.           |
| ぜひ コーヒーを飲みましょう。 | Nhất định cùng uống cà phê nhé. |
| ぜひ 来てください。      | Nhất định hãy đến nhé.          |
| ぜひ 留学したいです。     | Tôi rất muốn đi du học.         |
| ぜひ 恋人に会いたいです。   | Tôi rất muốn gặp người yêu.     |

# Tổng kết cực ngắn

| Mẫu          | Nghĩa             | Ví dụ       |
| ------------ | ----------------- | ----------- |
| `Vる`         | thể từ điển       | 食べる、読む、書く   |
| `Nができます`     | có thể làm N      | 日本語ができます    |
| `Vることができます`  | có thể làm V      | 泳ぐことができます   |
| `Vることができません` | không thể làm V   | 泳ぐことができません  |
| `趣味はNです`     | sở thích là N     | 趣味は料理です     |
| `趣味はVることです`  | sở thích là làm V | 趣味は本を読むことです |
| `Vる前に`       | trước khi V       | 寝る前に、本を読みます |
| `Nの前に`       | trước N           | 会議の前に       |
| `時間 + 前に`    | cách đây…         | 2年前に        |
| `なかなかVません`   | mãi mà không V    | バスがなかなか来ません |
| `ぜひVたいです`    | rất muốn V        | ぜひ日本へ行きたいです |
| `ぜひVてください`   | nhất định hãy V   | ぜひ来てください    |
