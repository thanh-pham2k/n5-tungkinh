# Minna no Nihongo Bài 6 — Tổng hợp + Bài tập trắc nghiệm

## 1. Tổng quan bài 6

| Nội dung            | Mẫu           | Nghĩa                |
| ------------------- | ------------- | -------------------- |
| Tân ngữ + động từ   | Nを Vます        | Làm V tác động vào N |
| Làm/chơi/tổ chức    | Nを します        | Làm/chơi/tổ chức N   |
| Hỏi làm gì          | なにを しますか      | Làm gì?              |
| Phủ định hoàn toàn  | なにも Vません      | Không làm gì cả      |
| Địa điểm hành động  | Địa điểmで Vます | Làm V ở địa điểm     |
| Hỏi ở đâu           | どこで Vますか      | Làm V ở đâu?         |
| Rủ rê lịch sự       | Vませんか         | Cùng làm V không?    |
| Đề nghị tích cực    | Vましょう         | Cùng làm V nhé       |
| Tiếp nhận thông tin | そうですか         | Vậy à                |

# 2. Danh từ を Động từ

## Công thức

```text
N を Vます
```

## Nghĩa

**Làm hành động V tác động vào danh từ N.**

Ví dụ:

| Câu          | Nghĩa              |
| ------------ | ------------------ |
| あさごはんを たべます。 | Ăn cơm sáng.       |
| ほんを よみます。    | Đọc sách.          |
| ジュースを のみます。  | Uống nước hoa quả. |
| えいがを みます。    | Xem phim.          |
| たばこを すいます。   | Hút thuốc.         |

## Ghi nhớ nhanh

Khi **động từ tác động trực tiếp vào danh từ**, dùng trợ từ **を**.

Ví dụ:

```text
ほんを よみます。
```

= Đọc sách.

Trong đó:

| Thành phần | Vai trò                 |
| ---------- | ----------------------- |
| ほん         | danh từ chịu tác động   |
| を          | trợ từ đánh dấu tân ngữ |
| よみます       | động từ                 |

---

# 3. Câu khẳng định với NをVます

## Công thức

```text
S は N を Vます。
```

## Nghĩa

**S làm V với N.**

Ví dụ:

| Câu                   | Nghĩa                        |
| --------------------- | ---------------------------- |
| ランさんは にほんごを べんきょうします。 | Bạn Lan học tiếng Nhật.      |
| わたしは たまごと やさいを たべました。 | Tôi đã ăn trứng và rau.      |
| ちちは まいにち たばこを すいます。   | Bố tôi hút thuốc mỗi ngày.   |
| きのう ダオさんは ほんを よみました。  | Hôm qua bạn Đào đã đọc sách. |
| けさ コーヒーを のみました。       | Sáng nay tôi đã uống cà phê. |

---

# 4. Câu phủ định với NをVません

## Công thức

```text
S は N を Vません。
S は N を Vませんでした。
```

## Nghĩa

| Mẫu     | Nghĩa          |
| ------- | -------------- |
| Vません    | Không làm V    |
| Vませんでした | Đã không làm V |

Ví dụ:

| Câu                 | Nghĩa                              |
| ------------------- | ---------------------------------- |
| ナムさんは たばこを すいません。   | Bạn Nam không hút thuốc.           |
| おさけを のみません。         | Tôi không uống rượu.               |
| にくを たべません。          | Tôi không ăn thịt.                 |
| けさ しんぶんを よみませんでした。  | Sáng nay tôi đã không đọc báo.     |
| きのう レポートを かきませんでした。 | Hôm qua tôi đã không viết báo cáo. |

---

# 5. Câu hỏi xác nhận với NをVますか

## Công thức

```text
N を Vますか。
N を Vましたか。
```

## Nghĩa

**Có làm V không? / Đã làm V chưa?**

Ví dụ:

| Câu               | Nghĩa                         |
| ----------------- | ----------------------------- |
| さしみを たべますか。       | Bạn có ăn sashimi không?      |
| コーヒーを のみますか。      | Bạn có uống cà phê không?     |
| きのう えいがを みましたか。   | Hôm qua bạn đã xem phim chưa? |
| にほんごを べんきょうしましたか。 | Bạn đã học tiếng Nhật chưa?   |

## Cách trả lời

| Câu hỏi      | Trả lời khẳng định | Trả lời phủ định |
| ------------ | ------------------ | ---------------- |
| さしみを たべますか。  | はい、たべます。           | いいえ、たべません。       |
| コーヒーを のみますか。 | はい、のみます。           | いいえ、のみません。       |
| えいがを みましたか。  | はい、みました。           | いいえ、みませんでした。     |

# 6. なにをVますか — Làm gì?

## Công thức

```text
なにを Vますか。
なにを Vましたか。
```

## Nghĩa

**Làm gì? / Đã làm gì?**

Ví dụ:

| Câu             | Nghĩa                 |
| --------------- | --------------------- |
| なにを たべますか。      | Bạn ăn gì?            |
| きのう なにを たべましたか。 | Hôm qua bạn đã ăn gì? |
| なにを のみますか。      | Bạn uống gì?          |
| なにを みましたか。      | Bạn đã xem gì?        |
| なにを べんきょうしますか。  | Bạn học gì?           |

Ví dụ trả lời:

```text
きのう なにを たべましたか。
にくと さかなを たべました。
```

= Hôm qua bạn đã ăn gì?
= Tôi đã ăn thịt và cá.

# 7. Nをします

## Công thức

```text
N を します
```

## Nghĩa

Tùy danh từ đi trước, **します** có thể dịch là **làm / chơi / tổ chức**.

---

## 7.1. Chơi N

Thường dùng với **môn thể thao**.

| Câu        | Nghĩa           |
| ---------- | --------------- |
| サッカーを します。 | Chơi bóng đá.   |
| テニスを します。  | Chơi tennis.    |
| やきゅうを します。 | Chơi bóng chày. |

---

## 7.2. Làm N

Thường dùng với **công việc / bài tập / việc nhà**.

| Câu         | Nghĩa         |
| ----------- | ------------- |
| しゅくだいを します。 | Làm bài tập.  |
| しごとを します。   | Làm việc.     |
| かじを します。    | Làm việc nhà. |

---

## 7.3. Tổ chức N

Thường dùng với **sự kiện / cuộc họp / bữa tiệc**.

| Câu         | Nghĩa         |
| ----------- | ------------- |
| パーティーを します。 | Tổ chức tiệc. |
| かいぎを します。   | Tổ chức họp.  |

---

## 7.4. Một số cụm đặc biệt

| Câu         | Nghĩa        |
| ----------- | ------------ |
| おはなみを します。  | Đi ngắm hoa. |
| ジョギングを します。 | Chạy bộ.     |

---

# 8. Phủ định hoàn toàn: なにもVません

## Công thức

```text
なにも Vません。
なにも Vませんでした。
```

## Nghĩa

**Không làm gì cả / đã không làm gì cả**

Ví dụ:

| Câu              | Nghĩa                       |
| ---------------- | --------------------------- |
| なにも たべません。       | Không ăn gì cả.             |
| なにも のみません。       | Không uống gì cả.           |
| なにも しません。        | Không làm gì cả.            |
| なにも みませんでした。     | Đã không xem gì cả.         |
| きのう なにも しませんでした。 | Hôm qua đã không làm gì cả. |

So sánh nhanh:

| Mẫu       | Nghĩa          |
| --------- | -------------- |
| なにを たべますか | Ăn gì?         |
| なにも たべません | Không ăn gì cả |

---

# 9. Địa điểmでVます

## Công thức

```text
Địa điểm で Vます
```

## Nghĩa

**Làm hành động V ở địa điểm nào đó.**

Ví dụ:

| Câu                   | Nghĩa                    |
| --------------------- | ------------------------ |
| スーパーで パンを かいます。       | Mua bánh mì ở siêu thị.  |
| がっこうで にほんごを べんきょうします。 | Học tiếng Nhật ở trường. |
| レストランで ひるごはんを たべます。   | Ăn trưa ở nhà hàng.      |
| うちで しゅくだいを します。       | Làm bài tập ở nhà.       |
| としょかんで ほんを よみます。      | Đọc sách ở thư viện.     |

## Ghi nhớ

| Danh từ phía trước        | Trợ từ |
| ------------------------- | ------ |
| Địa điểm di chuyển đến    | へ      |
| Địa điểm xảy ra hành động | で      |
| Phương tiện               | で      |
| Người đi cùng             | と      |
| Tân ngữ bị tác động       | を      |

---

# 10. どこでVますか — Làm ở đâu?

## Công thức

```text
どこで Vますか。
どこで Vましたか。
```

## Nghĩa

**Làm V ở đâu? / Đã làm V ở đâu?**

Ví dụ:

| Câu                  | Nghĩa                     |
| -------------------- | ------------------------- |
| どこで パンを かいましたか。      | Bạn đã mua bánh mì ở đâu? |
| どこで にほんごを べんきょうしますか。 | Bạn học tiếng Nhật ở đâu? |
| どこで ひるごはんを たべますか。    | Bạn ăn trưa ở đâu?        |
| どこで たばこを すいますか。      | Hút thuốc ở đâu?          |
| どこで えいがを みますか。       | Xem phim ở đâu?           |

Phân biệt:

| Mẫu           | Nghĩa      |
| ------------- | ---------- |
| どこへ いきますか     | Đi đâu?    |
| どこで べんきょうしますか | Học ở đâu? |

---

# 11. Vませんか — Rủ rê lịch sự

## Công thức

```text
Vませんか。
```

## Nghĩa

**Cùng làm V không?**

Dùng để mời/rủ người khác một cách lịch sự, mềm mại.

Ví dụ:

| Câu            | Nghĩa                    |
| -------------- | ------------------------ |
| えいがを みませんか。    | Cùng xem phim không?     |
| ひるごはんを たべませんか。 | Cùng ăn trưa không?      |
| サッカーを しませんか。   | Cùng chơi bóng đá không? |
| コーヒーを のみませんか。  | Cùng uống cà phê không?  |
| にほんへ いきませんか。   | Cùng đi Nhật không?      |

## Cách trả lời

| Đồng ý     | Từ chối           |
| ---------- | ----------------- |
| ええ、いいですね。  | すみません、ちょっと…。      |
| ええ、いきましょう。 | すみません、きょうは ちょっと…。 |

---

# 12. Vましょう — Cùng làm V nhé

## Công thức

```text
Vましょう。
```

## Nghĩa

**Cùng làm V nhé / Nào, làm V thôi.**

Dùng khi người nói **chủ động đề xuất mạnh hơn** hoặc **đáp lại lời rủ rê**.

Ví dụ:

| Câu            | Nghĩa              |
| -------------- | ------------------ |
| やすみましょう。       | Cùng nghỉ nhé.     |
| はじめましょう。       | Bắt đầu nào.       |
| いきましょう。        | Đi thôi.           |
| ひるごはんを たべましょう。 | Cùng ăn trưa nhé.  |
| ビールを のみましょう。   | Cùng uống bia nhé. |

So sánh:

| Mẫu   | Cảm giác                                 |
| ----- | ---------------------------------------- |
| Vませんか | Rủ rê nhẹ nhàng, lịch sự                 |
| Vましょう | Đề xuất tích cực hơn / hưởng ứng lời mời |

---

# 13. そうですね / そうですか / よ

## そうですか

Dùng khi người nghe nhận được thông tin mới.

```text
そうですか。
```

= Vậy à.

Ví dụ:

```text
きのう ホーグオムへ いきました。
そうですか。
```

= Hôm qua tôi đã đi Hồ Gươm.
= Vậy à.

---

## よ

Dùng để nhấn mạnh thông tin người nói nghĩ người nghe chưa biết.

```text
じてんしゃで きましたよ。
```

= Tôi đã đến bằng xe đạp đấy.

# 8. Phủ định hoàn toàn: なにもVません

## Công thức


text
なにも Vません。
なにも Vませんでした。


## Nghĩa

**Không làm gì cả / đã không làm gì cả**

Ví dụ:

| Câu              | Nghĩa                       |
| ---------------- | --------------------------- |
| なにも たべません。       | Không ăn gì cả.             |
| なにも のみません。       | Không uống gì cả.           |
| なにも しません。        | Không làm gì cả.            |
| なにも みませんでした。     | Đã không xem gì cả.         |
| きのう なにも しませんでした。 | Hôm qua đã không làm gì cả. |

| Mẫu       | Nghĩa          |
| --------- | -------------- |

# 11. Vませんか — Rủ rê lịch sự

## Công thức


text
Vませんか。


## Nghĩa

**Cùng làm V không?**

Dùng để mời/rủ người khác một cách lịch sự, mềm mại.

Ví dụ:

| Câu            | Nghĩa                    |
| -------------- | ------------------------ |
| えいがを みませんか。    | Cùng xem phim không?     |
| ひるごはんを たべませんか。 | Cùng ăn trưa không?      |
| サッカーを しませんか。   | Cùng chơi bóng đá không? |
| コーヒーを のみませんか。  | Cùng uống cà phê không?  |
| にほんへ いきませんか。   | Cùng đi Nhật không?      |

## Cách trả lời

| Đồng ý     | Từ chối           |
| ---------- | ----------------- |
| ええ、いいですね。  | すみません、ちょっと…。      |
| ええ、いきましょう。 | すみません、きょうは ちょっと…。 |

# 12. Vましょう — Cùng làm V nhé

## Công thức


text
Vましょう。


## Nghĩa

**Cùng làm V nhé / Nào, làm V thôi.**

Dùng khi người nói **chủ động đề xuất mạnh hơn** hoặc **đáp lại lời rủ rê**.

Ví dụ:

| Câu            | Nghĩa              |
| -------------- | ------------------ |
| やすみましょう。       | Cùng nghỉ nhé.     |
| はじめましょう。       | Bắt đầu nào.       |
| いきましょう。        | Đi thôi.           |
| ひるごはんを たべましょう。 | Cùng ăn trưa nhé.  |
| ビールを のみましょう。   | Cùng uống bia nhé. |

So sánh:

| Mẫu   | Cảm giác                                 |
| ----- | ---------------------------------------- |
| Vませんか | Rủ rê nhẹ nhàng, lịch sự                 |
| Vましょう | Đề xuất tích cực hơn / hưởng ứng lời mời |

# 13. そうですね / そうですか / よ

## そうですか

Dùng khi người nghe nhận được thông tin mới.


text

text

text

= Tôi đã đến bằng xe đạp đấy.

Kết quả ôn tập Bài 6

Bạn làm đúng 70/70 câu 🎉

Bạn đã nắm tốt các điểm chính:

NをVます: ほんを よみます
Nをします: サッカーを します
なにも + phủ định: なにも たべません
Địa điểmでVます: レストランで たべます
Vませんか: lời mời
Vましょう: cùng làm nhé
そうですか: vậy à

Bài 6 như vậy là ôn khá đầy đủ rồi.
