# 2. ～とき — Khi…

## Công thức

| Loại từ   | Cách nối với とき         | Ví dụ       |
| --------- | ----------------------- | ----------- |
| Tính từ い | いAdj + とき               | さびしいとき      |
| Tính từ な | なAdj + な + とき           | ひまなとき       |
| Danh từ   | N + の + とき              | 子どものとき      |
| Động từ   | V thể thông thường + とき | 寝るとき / 寝たとき |

## Nghĩa

```text
Khi A thì B
```

Ví dụ:

| Câu                     | Nghĩa                                     |
| ----------------------- | ----------------------------------------- |
| 子どものとき、タイさんの家へ遊びに行きました。 | Khi còn nhỏ, tôi thường đến nhà Tài chơi. |
| さびしいとき、彼氏に会いたいです。       | Khi cô đơn, tôi muốn gặp bạn trai.        |
| 寝るとき、電気を消してください。        | Khi đi ngủ, hãy tắt đèn.                  |
| わからないとき、先生に聞いてください。     | Khi không hiểu, hãy hỏi giáo viên.        |

# 3. Vるとき và Vたとき

Đây là phần rất quan trọng.

## 3.1. Vるとき

Dùng khi hành động V1 **chưa xảy ra / chưa hoàn thành** tại thời điểm làm V2.

Ví dụ:

| Câu                      | Nghĩa                                  |
| ------------------------ | -------------------------------------- |
| 寝るとき、「おやすみなさい」と言います。     | Khi đi ngủ, nói “chúc ngủ ngon”.       |
| 寝るとき、電気を消してください。         | Khi đi ngủ, hãy tắt đèn.               |
| ごはんを食べるとき、「いただきます」と言います。 | Khi ăn cơm, nói “いただきます”.              |
| うちへ帰るとき、アイスクリームを買いました。   | Khi về nhà, tôi mua kem trên đường về. |

Hiểu đơn giản:

```text
寝るとき = trước / trong lúc chuẩn bị ngủ
帰るとき = trên đường về, chưa về tới nơi
```

---

## 3.2. Vたとき

Dùng khi hành động V1 **đã xảy ra / đã hoàn thành** rồi mới làm V2.

Ví dụ:

| Câu                       | Nghĩa                                  |
| ------------------------- | -------------------------------------- |
| うちへ帰ったとき、「ただいま」と言います。     | Khi về đến nhà, nói “tôi về rồi”.      |
| 会社へ来たとき、「おはようございます」と言います。 | Khi đến công ty, nói “chào buổi sáng”. |
| ごはんを食べたとき、「ごちそうさま」と言います。  | Khi ăn xong, nói “ごちそうさま”.             |
| パリへ行ったとき、このドレスを買いました。     | Khi đã đến Paris, tôi mua cái váy này. |

Hiểu đơn giản:

```text
帰ったとき = sau khi đã về tới nơi
食べたとき = sau khi đã ăn xong
```

# 4. ～とき、どうしますか — Khi… thì làm thế nào?

## Công thức

```text
V / Adj / N + とき、どうしますか。
```

## Nghĩa

**Khi gặp tình huống đó thì làm thế nào?**

Ví dụ:

| Câu hỏi              | Nghĩa                                  |
| -------------------- | -------------------------------------- |
| お金が出ないとき、どうしますか。     | Khi tiền không ra thì làm thế nào?     |
| 現金がないとき、どうしますか。      | Khi không có tiền mặt thì làm thế nào? |
| 熱のとき、どうしますか。         | Khi bị sốt thì làm thế nào?            |
| パスポートをなくしたとき、どうしますか。 | Khi làm mất hộ chiếu thì làm thế nào?  |

Ví dụ trả lời:

| Câu             | Nghĩa                   |
| --------------- | ----------------------- |
| 黄色いボタンを押してください。 | Hãy bấm nút màu vàng.   |
| カードで払ってもいいです。   | Có thể trả bằng thẻ.    |
| すぐ病院へ行ってください。   | Hãy đến bệnh viện ngay. |
| 大使館へ行ってください。    | Hãy đến đại sứ quán.    |

# 5. Vると、V2 — Hễ cứ… thì…

## Công thức

```text
Vる + と、V2
```

## Nghĩa

**Hễ cứ V1 thì V2 xảy ra.**

Dùng khi V2 là kết quả tự nhiên / chắc chắn xảy ra sau V1.

## Thường dùng trong 3 trường hợp

| Trường hợp                    | Ví dụ           |
| ----------------------------- | --------------- |
| Chỉ dẫn máy móc               | ここを押すと、切符が出ます。  |
| Chỉ đường                     | 右へ曲がると、銀行があります。 |
| Hiện tượng / phản xạ tự nhiên | 勉強すると、眠くなります。   |

Ví dụ:

| Câu                  | Nghĩa                                          |
| -------------------- | ---------------------------------------------- |
| このボタンを押すと、切符が出ます。    | Hễ bấm nút này thì vé sẽ ra.                   |
| 左へ回すと、音が小さくなります。     | Hễ vặn sang trái thì âm thanh nhỏ đi.          |
| その信号を右へ曲がると、銀行があります。 | Hễ rẽ phải ở đèn tín hiệu đó thì có ngân hàng. |
| 勉強すると、眠くなります。        | Hễ học bài là buồn ngủ.                        |
| 彼女に会うと、手をつなぎたくなります。  | Hễ gặp cô ấy là muốn nắm tay.                  |

---

# 6. Lưu ý với Vると

Vế sau **không dùng ý chí / lời mời / mệnh lệnh**.

Nên dùng cho kết quả tự nhiên:

```text
このボタンを押すと、切符が出ます。
```

Không dùng kiểu:

```text
このボタンを押すと、行きましょう。❌
```

Vì **行きましょう** là lời mời, không phải kết quả tự nhiên.

# 7. Nが Adjです — Danh từ có tính chất…

## Công thức

```text
N が Adj です。
```

## Nghĩa

Dùng để miêu tả tính chất của danh từ một cách khách quan.

Ví dụ:

| Câu            | Nghĩa                     |
| -------------- | ------------------------- |
| あの女の人は顔がきれいです。 | Người phụ nữ kia mặt đẹp. |
| 音が小さいです。       | Âm thanh nhỏ.             |
| この料理は味がおいしいです。 | Món này vị ngon.          |
| 今日、天気が寒いです。    | Hôm nay thời tiết lạnh.   |
| 部屋がきれいです。      | Phòng sạch.               |
| 空が青いです。        | Bầu trời xanh.            |

Ghi nhớ nhanh:

```text
Danh từ + が + tính từ
```

Ví dụ rất hay gặp:

| Mẫu   | Nghĩa   |
| ----- | ------- |
| 背が高い  | Cao     |
| 鼻が長い  | Mũi dài |
| 髪が黒い  | Tóc đen |
| 顔がきれい | Mặt đẹp |

---

# 8. Nを + động từ di chuyển

## Công thức

```text
N を + động từ di chuyển
```

## Nghĩa

**Đi qua / di chuyển trên / băng qua N.**

Dùng với các động từ di chuyển như:

| Động từ | Nghĩa    |
| ------- | -------- |
| 歩きます    | đi bộ    |
| 散歩します   | đi dạo   |
| 渡ります    | băng qua |
| 曲がります   | rẽ       |
| 飛びます    | bay      |

Ví dụ:

| Câu            | Nghĩa                       |
| -------------- | --------------------------- |
| 道を渡ります。        | Băng qua đường.             |
| 公園を散歩します。      | Đi dạo trong công viên.     |
| 道の右を歩きます。      | Đi bộ bên phải đường.       |
| 飛行機が空を飛んでいます。  | Máy bay đang bay trên trời. |
| 橋を渡って、右へ曲がります。 | Qua cầu rồi rẽ phải.        |

Ghi nhớ:

| Ý nghĩa                 | Trợ từ |
| ----------------------- | ------ |
| Đi đến nơi nào          | へ / に  |
| Làm hành động ở đâu     | で      |
| Đi qua / di chuyển trên | を      |
