# 2. Thể ない là gì?

**Thể ない** là dạng phủ định thông thường của động từ.

| Lịch sự | Thông thường |
| ------- | ------------ |
| 食べません   | 食べない         |
| 行きません   | 行かない         |
| 書きません   | 書かない         |
| しません    | しない          |
| 来ません    | 来ない          |

Nghĩa chung: **không làm V**.

Ví dụ:

| Câu       | Nghĩa           |
| --------- | --------------- |
| ごはんを食べない。 | Không ăn cơm.   |
| 学校へ行かない。  | Không đi học.   |
| 本を読まない。   | Không đọc sách. |

# 3. Cách chia thể ない

## 3.1. Nhóm 1

Quy tắc:

```text
Âm trước ます thuộc cột い → chuyển sang cột あ + ない
```

| Vます  | Vない  | Nghĩa      |
| ---- | ---- | ---------- |
| 行きます | 行かない | không đi   |
| 書きます | 書かない | không viết |
| 読みます | 読まない | không đọc  |
| 話します | 話さない | không nói  |
| 取ります | 取らない | không lấy  |
| 呼びます | 呼ばない | không gọi  |
| 急ぎます | 急がない | không vội  |
| 帰ります | 帰らない | không về   |

### Trường hợp đặc biệt với âm い

Nếu trước ます là **い**, đổi thành **わ + ない**.

| Vます  | Vない  | Nghĩa          |
| ---- | ---- | -------------- |
| 言います | 言わない | không nói      |
| 吸います | 吸わない | không hút      |
| 払います | 払わない | không trả tiền |
| 会います | 会わない | không gặp      |

### Động từ đặc biệt

| Vます  | Vない |
| ---- | --- |
| あります | ない  |

Bỏ ます + ない

# 4. Vないでください — Xin đừng làm V

## Công thức

```text
Vないでください。
```

## Nghĩa

**Xin đừng làm V.**

Dùng để yêu cầu người khác **không làm việc gì đó**.

Ví dụ:

| Câu                | Nghĩa                     |
| ------------------ | ------------------------- |
| ここで たばこを吸わないでください。 | Xin đừng hút thuốc ở đây. |
| 心配しないでください。        | Xin đừng lo lắng.         |
| 私のことを忘れないでください。    | Xin đừng quên tôi.        |
| ここに車を止めないでください。    | Xin đừng đỗ xe ở đây.     |
| 本に書かないでください。       | Xin đừng viết vào sách.   |
| このボタンを押さないでください。   | Xin đừng bấm nút này.     |

# 5. Vなければなりません — Phải làm V

## Công thức

```text
Vない bỏ い + ければなりません。
```

## Nghĩa

**Phải làm V.**

Dùng khi việc đó là **bắt buộc**, không phụ thuộc vào ý muốn cá nhân.

Ví dụ:

| Câu                         | Nghĩa                                       |
| --------------------------- | ------------------------------------------- |
| レポートを書かなければなりません。           | Tôi phải viết báo cáo.                      |
| 薬を飲まなければなりません。              | Tôi phải uống thuốc.                        |
| 現金がありませんから、カードで払わなければなりません。 | Vì không có tiền mặt nên phải trả bằng thẻ. |
| 金曜日までに終わらなければなりません。         | Phải xong trước thứ Sáu.                    |
| 予約しなければなりません。               | Phải đặt chỗ trước.                         |
| 保険証を持って行かなければなりません。         | Phải mang thẻ bảo hiểm đi.                  |

Cách hiểu nhanh:

```text
書かない → 書かなければなりません
飲まない → 飲まなければなりません
しない → しなければなりません
来ない → 来なければなりません
```

# 6. Vなくてもいいです — Không cần làm V cũng được

## Công thức

```text
Vない bỏ い + くてもいいです。
```

## Nghĩa

**Không cần làm V cũng được.**

Dùng để nói việc đó **không bắt buộc**.

Ví dụ:

| Câu                 | Nghĩa                                       |
| ------------------- | ------------------------------------------- |
| 明日 会社へ来なくてもいいです。    | Ngày mai không cần đến công ty cũng được.   |
| 心配しなくてもいいです。        | Không cần lo lắng cũng được.                |
| 傘を持って行かなくてもいいです。    | Không cần mang ô cũng được.                 |
| 金曜日までに終わらなくてもいいですか。 | Không cần xong trước thứ Sáu cũng được chứ? |
| 薬を飲まなくてもいいですか。      | Không cần uống thuốc cũng được chứ?         |
| 靴を脱がなくてもいいです。       | Không cần cởi giày cũng được.               |
| 名前を書かなくてもいいです。      | Không cần viết tên cũng được.               |

Cách hiểu nhanh:

```text
書かない → 書かなくてもいいです
飲まない → 飲まなくてもいいです
しない → しなくてもいいです
来ない → 来なくてもいいです
```

# 7. Đưa tân ngữ lên làm chủ đề với は

## Mẫu gốc

```text
Nを Vます。
```

Ví dụ:

```text
荷物をここに置かないでください。
```

= Xin đừng đặt hành lý ở đây.

## Mẫu nhấn mạnh

```text
Nは Vます。
```

Ví dụ:

```text
荷物はここに置かないでください。
```

= Hành lý thì xin đừng đặt ở đây.

Mẫu này dùng để **nhấn mạnh danh từ**.

Ví dụ:

| Câu               | Nghĩa                           |
| ----------------- | ------------------------------- |
| 昼ごはんは レストランで食べます。 | Cơm trưa thì tôi ăn ở nhà hàng. |
| 車は ここに止めてください。    | Xe ô tô thì hãy đỗ ở đây.       |
| 絵は 鉛筆で描きます。       | Tranh thì tôi vẽ bằng bút chì.  |
| レポートは ファックスで送ります。 | Báo cáo thì tôi gửi bằng fax.   |

---

# 8. までに — trước hạn / chậm nhất là

## Công thức

```text
Thời gian + までに + Vます。
```

## Nghĩa

**Làm V trước thời điểm đó / chậm nhất là thời điểm đó.**

Dùng cho **deadline / hạn chót**.

Ví dụ:

| Câu                      | Nghĩa                                 |
| ------------------------ | ------------------------------------- |
| 午後5時までに レポートを出してください。    | Hãy nộp báo cáo trước 5 giờ chiều.    |
| 10時までに 会議が終わります。         | Cuộc họp sẽ kết thúc trước 10 giờ.    |
| 今週の土曜日までに 本を返さなければなりません。 | Phải trả sách trước thứ Bảy tuần này. |
| 11時までに 寝てください。           | Hãy đi ngủ trước 11 giờ.              |
| 9時までに 帰らなければなりません。       | Phải về trước 9 giờ.                  |

## Phân biệt まで và までに

| Mẫu | Nghĩa                    | Ví dụ         |
| --- | ------------------------ | ------------- |
| まで  | đến / cho đến            | 8時から5時まで働きます。 |
| までに | trước hạn / chậm nhất là | 5時までに出してください。 |

Ghi nhớ:

```text
まで = kéo dài đến mốc đó
までに = hoàn thành trước mốc đó
```

# Tổng kết cực ngắn

| Mẫu          | Nghĩa                    | Ví dụ        |
| ------------ | ------------------------ | ------------ |
| `Vない`        | không làm V              | 食べない         |
| `Vないでください`   | xin đừng làm V           | 吸わないでください    |
| `Vなければなりません` | phải làm V               | 書かなければなりません  |
| `Vなくてもいいです`  | không cần làm V          | 来なくてもいいです    |
| `Nは Vます`     | N thì… / nhấn mạnh N     | 荷物は置かないでください |
| `時間までに Vます`  | làm V trước hạn          | 5時までに出してください |
| `まで`         | đến / cho đến            | 8時から5時まで     |
| `までに`        | trước hạn / chậm nhất là | 5時までに        |
