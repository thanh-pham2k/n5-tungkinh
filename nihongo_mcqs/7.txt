# Minna no Nihongo Bài 7 — Tổng hợp + Bài tập trắc nghiệm

## 1. Tổng quan bài 7

| Nội dung              | Mẫu                  | Nghĩa                  |
| --------------------- | -------------------- | ---------------------- |
| Công cụ / phương tiện | Nで Vます               | Làm V bằng N           |
| Hỏi bằng gì           | なんで Vますか             | Làm V bằng gì?         |
| Hỏi cách nói          | Aは Bごで なんですか         | A trong tiếng B là gì? |
| Cho / tặng            | Aは Bに Nを あげます        | A cho B N              |
| Nhận                  | Bは Aに / から Nを もらいます  | B nhận N từ A          |
| Hỏi cho ai            | だれに Nを あげますか         | Cho ai N?              |
| Hỏi nhận từ ai        | だれに / だれから Nを もらいますか | Nhận từ ai?            |
| Đã làm chưa           | もう Vましたか             | Đã làm V chưa?         |
| Chưa làm              | まだです / まだ Vていません     | Vẫn chưa               |

# 2. NでVます — Làm bằng công cụ / cách thức

## Công thức

```text
N で Vます
```

## Nghĩa

**Làm hành động V bằng N.**

N có thể là:

| Loại danh từ         | Ví dụ            | Nghĩa                    |
| -------------------- | ---------------- | ------------------------ |
| Công cụ              | はし, ナイフ, はさみ, ペン | đũa, dao, kéo, bút       |
| Phương tiện          | バス, じてんしゃ, ひこうき  | xe buýt, xe đạp, máy bay |
| Cách thức / ngôn ngữ | にほんご, えいご        | tiếng Nhật, tiếng Anh    |

Ví dụ:

| Câu              | Nghĩa                       |
| ---------------- | --------------------------- |
| はしで ごはんを たべます。   | Ăn cơm bằng đũa.            |
| はさみで かみを きります。   | Cắt giấy bằng kéo.          |
| ナイフで やさいを きります。  | Cắt rau bằng dao.           |
| にほんごで てがみを かきます。 | Viết thư bằng tiếng Nhật.   |
| でんわで はなします。      | Nói chuyện bằng điện thoại. |

# 3. なんでVますか — Làm bằng gì?

## Công thức

```text
なんで Vますか。
なんで Vましたか。
```

## Nghĩa

**Làm V bằng gì?**

Ví dụ:

| Câu hỏi           | Trả lời     |
| ----------------- | ----------- |
| なんで ごはんを たべますか。   | はしで たべます。   |
| なんで しゃしんを とりますか。  | けいたいで とります。 |
| なんで おんがくを ききますか。  | ラジオで ききます。  |
| なんで レポートを かきましたか。 | えいごで かきました。 |

Lưu ý: **なんで** cũng có thể nghĩa là “tại sao”, tùy ngữ cảnh.

# 4. “A trong tiếng B là gì?”

## Công thức

```text
A は Bごで なんですか。
```

## Nghĩa

**A trong tiếng B là gì?**

Ví dụ:

| Câu                            | Nghĩa                                    |
| ------------------------------ | ---------------------------------------- |
| 「Cảm ơn」は にほんごで なんですか。         | “Cảm ơn” trong tiếng Nhật là gì?         |
| 「Xin chào」は えいごで なんですか。        | “Xin chào” trong tiếng Anh là gì?        |
| “I love you” は ベトナムごで なんですか。   | “I love you” trong tiếng Việt là gì?     |
| 「Chào buổi sáng」は にほんごで なんですか。 | “Chào buổi sáng” trong tiếng Nhật là gì? |

Ví dụ trả lời:

```text
「Cảm ơn」は にほんごで ありがとうです。
```

= “Cảm ơn” trong tiếng Nhật là ありがとう.

# 5. AはBにNをあげます — Cho / tặng

## Công thức

```text
A は B に N を あげます
```

## Nghĩa

**A cho / tặng B cái N.**

Trong đó:

| Thành phần | Vai trò      |
| ---------- | ------------ |
| A          | người cho    |
| B          | người nhận   |
| N          | vật được cho |
| あげます       | cho / tặng   |

Ví dụ:

| Câu                     | Nghĩa                               |
| ----------------------- | ----------------------------------- |
| わたしは ははに はなを あげます。      | Tôi tặng hoa cho mẹ.                |
| ナムさんは ランさんに はなを あげました。  | Nam đã tặng hoa cho Lan.            |
| ははは ちちに シャツを あげました。     | Mẹ đã tặng áo sơ mi cho bố.         |
| リンさんは こいびとに てがみを かきました。 | Chị Linh đã viết thư cho người yêu. |

---

## Một số động từ “cho đi”

| Động từ   | Nghĩa         |
| --------- | ------------- |
| あげます      | cho / tặng    |
| かします      | cho mượn      |
| おしえます     | dạy / chỉ cho |
| おくります     | gửi cho       |
| かきます      | viết cho      |
| でんわを かけます | gọi điện cho  |

Ví dụ:

| Câu                         | Nghĩa                               |
| --------------------------- | ----------------------------------- |
| たなかせんせいは ナムさんに にほんごを おしえます。 | Thầy Tanaka dạy tiếng Nhật cho Nam. |
| ナムさんは アンさんに ほんを かしました。      | Nam đã cho An mượn sách.            |
| わたしは キムラさんに レポートを おくります。    | Tôi gửi báo cáo cho chị Kimura.     |
| わたしは ははに でんわを かけます。         | Tôi gọi điện cho mẹ.                |

# 6. BはAに / からNをもらいます — Nhận từ ai

## Công thức

```text
B は A に N を もらいます
B は A から N を もらいます
```

## Nghĩa

**B nhận N từ A.**

Trong đó:

| Thành phần | Vai trò       |
| ---------- | ------------- |
| B          | người nhận    |
| A          | người cho     |
| N          | vật được nhận |
| もらいます      | nhận          |

Ví dụ:

| Câu                          | Nghĩa                                |
| ---------------------------- | ------------------------------------ |
| わたしは ははに はなを もらいました。         | Tôi đã nhận hoa từ mẹ.               |
| トランさんは やまだせんせいに にほんごを ならいます。 | Trang học tiếng Nhật từ thầy Yamada. |
| わたしは タロウさんに おかねを かりました。      | Tôi đã vay tiền từ Taro.             |
| わたしは がくせいから プレゼントを もらいました。   | Tôi đã nhận quà từ học sinh.         |
| わたしは ぎんこうから おかねを かりました。      | Tôi đã vay tiền từ ngân hàng.        |

Lưu ý:

| Người cho là…                          | Dùng   |
| -------------------------------------- | ------ |
| Người cụ thể                           | に / から |
| Tổ chức / công ty / trường / ngân hàng | から     |

---

## Một số động từ “nhận”

| Động từ | Nghĩa      |
| ------- | ---------- |
| もらいます   | nhận       |
| ならいます   | học từ ai  |
| かります    | vay / mượn |

# 8. もうVましたか — Đã làm chưa?

## Công thức

```text
もう Vましたか。
```

## Nghĩa

**Đã làm V chưa?**

Ví dụ:

| Câu                  | Nghĩa                        |
| -------------------- | ---------------------------- |
| もう ひるごはんを たべましたか。    | Đã ăn trưa chưa?             |
| もう レポートを かきましたか。     | Đã viết báo cáo chưa?        |
| もう にほんごの ほんを かいましたか。 | Đã mua sách tiếng Nhật chưa? |
| もう しゅくだいを しましたか。     | Đã làm bài tập chưa?         |

## Cách trả lời

| Trường hợp | Mẫu trả lời    |
| ---------- | -------------- |
| Đã rồi     | はい、もう Vました。    |
| Chưa       | いいえ、まだです。      |
| Chưa làm   | いいえ、まだ Vていません。 |

Ví dụ:

| Câu hỏi           | Trả lời             |
| ----------------- | ------------------- |
| もう ひるごはんを たべましたか。 | はい、もう たべました。        |
| もう くすりを のみましたか。   | いいえ、まだです。これから のみます。 |
| もう しゅくだいを しましたか。  | いいえ、まだ していません。      |

Lưu ý:

```text
いいえ、まだです。✅
いいえ、まだ Vていません。✅
いいえ、Vませんでした。❌ nếu muốn nói “chưa”
```
