# Minna no Nihongo Bài 20 — Tổng hợp + Bài tập trắc nghiệm

## 1. Tổng quan bài 20

| Nội dung                 | Mẫu / Ý chính             | Nghĩa             |
| ------------------------ | ------------------------- | ----------------- |
| Thể lịch sự              | です / ます                   | Cách nói lịch sự  |
| Thể thông thường         | Dạng ngắn                 | Cách nói thân mật |
| Động từ                  | Vます → Vる                  | Ăn → 食べる          |
| Động từ phủ định         | Vません → Vない                | Không ăn → 食べない   |
| Động từ quá khứ          | Vました → Vた                 | Đã ăn → 食べた       |
| Động từ phủ định quá khứ | Vませんでした → Vなかった           | Đã không ăn       |
| Tính từ い                | Bỏ です                     | おいしいです → おいしい     |
| Tính từ な / danh từ      | です → だ                    | 元気です → 元気だ        |
| Câu hỏi thân mật         | Bỏ か, lên giọng           | 行く？               |
| Hội thoại thân mật       | Có thể lược を / は / が / へ | ごはん食べる？           |

# 2. Thể thông thường là gì?

**Thể thông thường** là cách nói ngắn gọn, thân mật, thường dùng với:

| Dùng với                | Ví dụ   |
| ----------------------- | ------- |
| Bạn bè                  | もう食べた？  |
| Người thân              | 今日、忙しい？ |
| Người dưới / người quen | これ、見て。  |
| Hội thoại đời sống      | うん、行く。  |

Không nên dùng với:

| Không nên dùng với  | Nên dùng    |
| ------------------- | ----------- |
| Người mới gặp       | Thể lịch sự |
| Cấp trên            | Thể lịch sự |
| Khách hàng          | Thể lịch sự |
| Sự kiện trang trọng | Thể lịch sự |

Ví dụ:

| Lịch sự   | Thông thường |
| --------- | ------------ |
| ごはんを食べます。 | ごはんを食べる。     |
| 日本へ行きました。 | 日本へ行った。      |
| 今日は寒いです。  | 今日は寒い。       |
| 元気です。     | 元気だ。         |

# 3. Thể thông thường của động từ

## 3.1. Bảng chuyển đổi

| Thể lịch sự | Thể thông thường | Ví dụ             |
| ----------- | ---------------- | ----------------- |
| Vます         | Vる / thể từ điển | 食べます → 食べる        |
| Vません        | Vない              | 食べません → 食べない      |
| Vました        | Vた               | 食べました → 食べた       |
| Vませんでした     | Vなかった            | 食べませんでした → 食べなかった |

---

## 3.2. Ví dụ động từ

| Lịch sự  | Thông thường | Nghĩa        |
| -------- | ------------ | ------------ |
| 行きます     | 行く           | đi           |
| 行きません    | 行かない         | không đi     |
| 行きました    | 行った          | đã đi        |
| 行きませんでした | 行かなかった       | đã không đi  |
| 食べます     | 食べる          | ăn           |
| 食べません    | 食べない         | không ăn     |
| 食べました    | 食べた          | đã ăn        |
| 食べませんでした | 食べなかった       | đã không ăn  |
| します      | する           | làm          |
| しません     | しない          | không làm    |
| しました     | した           | đã làm       |
| しませんでした  | しなかった        | đã không làm |

Ví dụ câu:

| Lịch sự            | Thông thường     |
| ------------------ | ---------------- |
| 昨日、友達と遊びました。       | 昨日、友達と遊んだ。       |
| 週末、アメリカへ出張します。     | 週末、アメリカへ出張する。    |
| 彼女の気持ちがわかりません。     | 彼女の気持ちがわからない。    |
| 昨日の夜、ごはんを食べませんでした。 | 昨日の夜、ごはんを食べなかった。 |

# 4. Thể thông thường của tính từ い

## Quy tắc chính

Với **tính từ い**, chỉ cần **bỏ です**.

| Thể lịch sự | Thể thông thường |
| ----------- | ---------------- |
| おいしいです      | おいしい             |
| おいしくないです    | おいしくない           |
| おいしかったです    | おいしかった           |
| おいしくなかったです  | おいしくなかった         |

Ví dụ:

| Lịch sự  | Thông thường | Nghĩa         |
| -------- | ------------ | ------------- |
| 寒いです     | 寒い           | lạnh          |
| 寒くないです   | 寒くない         | không lạnh    |
| 寒かったです   | 寒かった         | đã lạnh       |
| 寒くなかったです | 寒くなかった       | đã không lạnh |
| 楽しいです    | 楽しい          | vui           |
| 楽しかったです  | 楽しかった        | đã vui        |

Ví dụ câu:

| Lịch sự           | Thông thường    |
| ----------------- | --------------- |
| 今日は寒いです。          | 今日は寒い。          |
| 昨日のパーティーは楽しかったです。 | 昨日のパーティーは楽しかった。 |
| この料理はおいしいです。      | この料理はおいしい。      |
| 先週、忙しくなかったです。     | 先週、忙しくなかった。     |

# 7. Thể thông thường trong mẫu ngữ pháp đã học

## 7.1. Một số mẫu thường gặp

| Lịch sự    | Thông thường | Nghĩa               |
| ---------- | ------------ | ------------------- |
| Vなければなりません | Vなければならない    | Phải làm V          |
| Vています      | Vている         | Đang làm V          |
| Vてはいけません   | Vてはいけない      | Không được làm V    |
| Vてもいいです    | Vてもいい        | Được phép làm V     |
| Vることができます  | Vることができる     | Có thể làm V        |
| Vたことがあります  | Vたことがある      | Đã từng làm V       |
| Vたり、Vたりします | Vたり、Vたりする    | Làm V này V kia     |
| ～になります     | ～になる         | Trở nên / trở thành |

Ví dụ:

| Lịch sự             | Thông thường       |
| ------------------- | ------------------ |
| ごはんを食べています。         | ごはんを食べている。         |
| 1日に2回薬を飲まなければなりません。 | 1日に2回薬を飲まなければならない。 |
| 写真を撮ってもいいです。        | 写真を撮ってもいい。         |
| ここでたばこを吸ってはいけません。   | ここでたばこを吸ってはいけない。   |
| カラオケで歌うことができません。    | カラオケで歌うことができない。    |
| デートしたことがありません。      | デートしたことがない。        |
| 天気が寒くなります。          | 天気が寒くなる。           |

# 8. Thể thông thường trong câu ghép

Khi dùng câu ghép, thường chuyển **cả hai vế** sang thể thông thường.

## 8.1. Vì… nên…

| Lịch sự                  | Thông thường          |
| ------------------------ | --------------------- |
| 歌が下手ですから、カラオケが好きじゃありません。 | 歌が下手だから、カラオケが好きじゃない。  |
| 今朝、雨でしたから、遅れました。         | 今朝、雨だったから、遅れた。        |
| 先週、病気でしたから、旅行に行きませんでした。  | 先週、病気だったから、旅行に行かなかった。 |

## 8.2. Nhưng…

Trong hội thoại, **が** thường đổi thành **けど**.

| Lịch sự              | Thông thường       |
| -------------------- | ------------------ |
| 日本語は難しいですが、おもしろいです。  | 日本語は難しいけど、おもしろい。   |
| カレーは辛いですが、おいしいです。    | カレーは辛いけど、おいしい。     |
| 残念ですが、資料を持って来ませんでした。 | 残念だけど、資料を持って来なかった。 |

# 9. Câu hỏi ở thể thông thường

## Quy tắc

| Lịch sự | Thông thường |
| ------- | ------------ |
| Vますか    | Vる？          |
| Adjですか  | Adj？         |
| Nですか    | N？           |
| なAdjですか | なAdj？        |

Khi nói: **lên giọng cuối câu**.
Khi viết: thêm **？**.

Ví dụ:

| Lịch sự      | Thông thường     |
| ------------ | ---------------- |
| 映画を見に行きますか。  | 映画を見に行く？         |
| コーヒーを飲みませんか。 | コーヒーを飲まない？       |
| ごはんを食べましたか。  | ごはんを食べた？         |
| 元気ですか。       | 元気？              |
| これは何ですか。     | これ、何？            |
| 泳ぐことができますか。  | 泳げる？ / 泳ぐことができる？ |

# 10. Cách trả lời thân mật

| Lịch sự   | Thân mật |
| --------- | -------- |
| はい        | うん       |
| いいえ       | ううん      |
| はい、あります   | うん、ある    |
| いいえ、ありません | ううん、ない   |
| はい、行きます   | うん、行く    |
| いいえ、行きません | ううん、行かない |

Ví dụ hội thoại:

```text
A: 今、暇？
B: うん、暇。
```

= Bây giờ rảnh không?
= Ừ, rảnh.

```text
A: ごはん食べた？
B: ううん、まだ。
```

= Ăn cơm chưa?
= Chưa.

```text
A: 何か飲む？
B: うん、飲む。
```

= Uống gì không?
= Ừ, uống.

# 11. Lược bỏ trợ từ trong hội thoại

Trong hội thoại thân mật, các trợ từ sau thường được lược bỏ:

| Trợ từ | Có thể lược |
| ------ | ----------- |
| を      | Có          |
| は      | Có          |
| が      | Có          |
| へ      | Có          |

Các trợ từ sau ít khi lược:

| Trợ từ | Ghi chú |
| ------ | ------- |
| に      | Ít lược |
| で      | Ít lược |
| と      | Ít lược |

Ví dụ:

| Đầy đủ       | Rút gọn thân mật |
| ------------ | ---------------- |
| ごはんを食べる？     | ごはん食べる？          |
| 明日、東京へ行く？    | 明日、東京行く？         |
| 時間がある？       | 時間ある？            |
| このりんごがおいしいね。 | このりんご、おいしいね。     |

---

# 12. Rút gọn trong mẫu Vています / Vてください

## 12.1. Vています → Vている → Vてる

| Đầy đủ   | Rút gọn      |
| -------- | ------------ |
| 食べています   | 食べている / 食べてる |
| 読んでいます   | 読んでいる / 読んでる |
| 何をしていますか | 何してる？        |

Ví dụ:

| Lịch sự     | Thân mật |
| ----------- | -------- |
| 母は料理をしています。 | 母、料理してる。 |
| 何をしていますか。   | 何してる？    |

## 12.2. Vてください → Vて

| Lịch sự      | Thân mật |
| ------------ | -------- |
| 見てください       | 見て       |
| 待ってください      | 待って      |
| 勉強してください     | 勉強して     |
| エアコンをつけてください | エアコンつけて  |

Ví dụ:

```text
これ、見て。
```

= Nhìn cái này đi.

```text
ちょっと待って。
```

= Đợi chút.

# Tổng kết cực ngắn

| Mẫu               | Nghĩa              | Ví dụ             |
| ----------------- | ------------------ | ----------------- |
| `Vます → Vる`        | làm V              | 食べます → 食べる        |
| `Vません → Vない`      | không làm V        | 食べません → 食べない      |
| `Vました → Vた`       | đã làm V           | 食べました → 食べた       |
| `Vませんでした → Vなかった` | đã không làm V     | 食べませんでした → 食べなかった |
| `いAdjです → いAdj`   | tính từ い thể ngắn | 寒いです → 寒い         |
| `なAdjです → なAdjだ`  | tính từ な thể ngắn | 元気です → 元気だ        |
| `Nです → Nだ`        | danh từ thể ngắn   | 学生です → 学生だ        |
| `ですか / ますか → ?`   | câu hỏi thân mật   | 行きますか → 行く？       |
| `はい / いいえ`        | thân mật           | うん / ううん          |
| `Vています → Vてる`     | đang làm V         | 食べています → 食べてる     |
| `Vてください → Vて`     | hãy làm V          | 見てください → 見て       |
| `が → けど`          | nhưng, thân mật    | 難しいけど、おもしろい       |
