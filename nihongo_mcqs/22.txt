| Bổ nghĩa cho danh từ         | Từ / cụm từ + N        | Làm rõ nghĩa cho N |

# 2. Bổ nghĩa cho danh từ là gì?

**Bổ nghĩa** là phần đứng trước danh từ để làm rõ danh từ đó.

Ví dụ với **車**:

| Cụm     | Nghĩa             |
| ------- | ----------------- |
| 私の車     | Xe của tôi        |
| 新しい車    | Xe mới            |
| きれいな車   | Xe đẹp            |
| きのう買った車 | Xe đã mua hôm qua |

Trong tiếng Nhật:

```text
Bổ nghĩa + Danh từ
```

Khác tiếng Việt:

```text
Danh từ + bổ nghĩa
```

Ví dụ:

| Tiếng Việt               | Tiếng Nhật |
| ------------------------ | ---------- |
| Cái mũ màu xanh          | 青い帽子       |
| Cái mũ đẹp               | きれいな帽子     |
| Cái mũ của tôi           | 私の帽子       |
| Cái mũ tôi đã mua ở Nhật | 日本で買った帽子   |

# 3. Các kiểu bổ nghĩa cơ bản

## 3.1. Tính từ い + danh từ

```text
いAdj + N
```

Ví dụ:

| Cụm    | Nghĩa           |
| ------ | --------------- |
| 青い帽子   | Cái mũ màu xanh |
| おいしい料理 | Món ăn ngon     |
| 高い時計   | Đồng hồ đắt     |
| 小さい家   | Ngôi nhà nhỏ    |

---

## 3.2. Tính từ な + danh từ

```text
なAdj + N
```

Ví dụ:

| Cụm    | Nghĩa                |
| ------ | -------------------- |
| きれいな帽子 | Cái mũ đẹp           |
| 便利な町   | Thành phố tiện lợi   |
| 親切な人   | Người tốt bụng       |
| 有名な学校  | Trường học nổi tiếng |

---

## 3.3. Danh từ + の + danh từ

```text
N1 の N2
```

Ví dụ:

| Cụm   | Nghĩa              |
| ----- | ------------------ |
| 私の帽子  | Cái mũ của tôi     |
| 日本の料理 | Món ăn Nhật        |
| 先生の本  | Sách của giáo viên |
| 友達の家  | Nhà của bạn        |

# 4. Mệnh đề bổ nghĩa cho danh từ

## Công thức chính

```text
V thể thông thường + N
```

## Nghĩa

**N mà V / N được V / N đang V / N đã V**

Động từ trước danh từ phải chia về **thể thông thường**.

| Thể   | Ví dụ   | Nghĩa           |
| ----- | ------- | --------------- |
| Vる    | 住む家     | Nhà sẽ/đang ở   |
| Vない   | 行かない人   | Người không đi  |
| Vた    | 買った帽子   | Cái mũ đã mua   |
| Vなかった | 行かなかった所 | Nơi đã không đi |

---

## Ví dụ

| Cụm          | Nghĩa                           |
| ------------ | ------------------------------- |
| 私が愛している女の人   | Người phụ nữ tôi yêu            |
| 日本で買った帽子     | Cái mũ tôi đã mua ở Nhật        |
| タイさんから借りた雑誌  | Tạp chí tôi đã mượn từ Tài      |
| コーヒーを飲んでいる人  | Người đang uống cà phê          |
| 初めて彼女に会った所   | Nơi tôi gặp cô ấy lần đầu       |
| 日本語を勉強したセンター | Trung tâm tôi đã học tiếng Nhật |

---

# 5. Lưu ý về chủ ngữ trong mệnh đề bổ nghĩa

Trong mệnh đề bổ nghĩa, chủ ngữ phụ thường dùng **が**, không dùng **は**.

Ví dụ:

```text
私が買った帽子
```

= Cái mũ tôi đã mua.

Không nên nói:

```text
私は買った帽子 ❌
```

Ví dụ khác:

| Đúng         | Nghĩa                         |
| ------------ | ----------------------------- |
| 母が作った料理      | Món ăn mẹ nấu                 |
| 父が作ったいす      | Cái ghế bố làm                |
| ミラーさんが住んでいる家 | Ngôi nhà anh Miller đang sống |
| 友達が書いた手紙     | Lá thư bạn viết               |

---

# 6. Mệnh đề bổ nghĩa làm chủ ngữ

## Công thức

```text
[Mệnh đề bổ nghĩa + N] は / が + vị ngữ
```

Ví dụ:

| Câu                        | Nghĩa                                       |
| -------------------------- | ------------------------------------------- |
| 日本で買った帽子はとてもきれいです。         | Cái mũ tôi mua ở Nhật rất đẹp.              |
| 日本語を勉強したセンターはとても有名です。      | Trung tâm tôi học tiếng Nhật rất nổi tiếng. |
| 青いシャツを着て、眼鏡をかけている人は佐藤さんです。 | Người mặc áo xanh, đeo kính là anh Sato.    |
| ミラーさんが住んでいる家はきれいです。        | Ngôi nhà anh Miller đang sống rất đẹp.      |

---

# 7. Mệnh đề bổ nghĩa làm vị ngữ

## Công thức

```text
N は [mệnh đề bổ nghĩa + N] です
```

Ví dụ:

| Câu                 | Nghĩa                                        |
| ------------------- | -------------------------------------------- |
| 私の彼女は髪が黒くて、背が高い人です。 | Bạn gái tôi là người tóc đen và cao.         |
| 私の先生はコーヒーを飲んでいる人です。 | Giáo viên của tôi là người đang uống cà phê. |
| それは父が作ったいすです。       | Đó là cái ghế bố tôi làm.                    |
| 山田さんは歌っている人です。      | Anh Yamada là người đang hát.                |

---

# 8. Mệnh đề bổ nghĩa làm tân ngữ

## Công thức

```text
[Mệnh đề bổ nghĩa + N] を Vます
```

Ví dụ:

| Câu                     | Nghĩa                                    |
| ----------------------- | ---------------------------------------- |
| チーさんはナムさんから借りた本をなくしました。 | Chi đã làm mất cuốn sách mượn từ Nam.    |
| 私は初めて夫に会った所を忘れました。      | Tôi đã quên nơi lần đầu gặp chồng.       |
| 私は母がきのう作ったケーキを食べたいです。   | Tôi muốn ăn cái bánh mẹ làm hôm qua.     |
| 私は父が作った家に住みたいです。        | Tôi muốn sống trong ngôi nhà bố tôi xây. |

---

# 9. Mệnh đề bổ nghĩa vừa làm chủ ngữ vừa làm vị ngữ

Ví dụ:

```text
きのう私が捨てたシャツは、イエンさんが着ているシャツです。
```

= Cái áo tôi vứt hôm qua là cái áo Yến đang mặc.

```text
けさスーパーで会った人は、Thanhさんと結婚する人です。
```

= Người tôi gặp ở siêu thị sáng nay là người sẽ kết hôn với chị Thanh.

---

# 10. Các danh từ thường đi với mệnh đề bổ nghĩa

## 10.1. Vる時間

```text
Vる + 時間
```

= Thời gian để làm V.

Ví dụ:

| Câu              | Nghĩa                            |
| ---------------- | -------------------------------- |
| 新聞を読む時間がありません。   | Tôi không có thời gian đọc báo.  |
| 休む時間がほしいです。      | Tôi muốn có thời gian nghỉ ngơi. |
| 日本語を勉強する時間があります。 | Tôi có thời gian học tiếng Nhật. |

---

## 10.2. Vる約束

```text
Vる + 約束
```

= Cuộc hẹn / lời hứa làm V.

Ví dụ:

| Câu                 | Nghĩa                                    |
| ------------------- | ---------------------------------------- |
| 今晩、彼女と映画を見る約束があります。 | Tối nay tôi có hẹn xem phim với bạn gái. |
| 友達とごはんを食べる約束があります。  | Tôi có hẹn ăn cơm với bạn.               |
| 先生に会う約束があります。       | Tôi có hẹn gặp giáo viên.                |

---

## 10.3. Vる用事

```text
Vる + 用事
```

= Việc bận phải làm V.

Ví dụ:

| Câu              | Nghĩa                           |
| ---------------- | ------------------------------- |
| 週末、東京へ行く用事があります。 | Cuối tuần tôi có việc đi Tokyo. |
| 銀行へ行く用事があります。    | Tôi có việc đi ngân hàng.       |
| 母に電話する用事があります。   | Tôi có việc gọi điện cho mẹ.    |

---

# 11. Vましょうか — Cùng làm nhé

## Công thức

```text
Vます bỏ ます + ましょうか
```

## Nghĩa trong bài 22

**Chúng ta cùng V nhé / cùng làm V nhé.**

Ví dụ:

| Câu                  | Nghĩa                         |
| -------------------- | ----------------------------- |
| 今日は一緒に帰りましょうか。       | Hôm nay cùng về nhé.          |
| 今晩、映画を見に行きましょうか。     | Tối nay cùng đi xem phim nhé. |
| 荷物が重いですね。一緒に持ちましょうか。 | Đồ nặng nhỉ. Cùng nâng nhé.   |

Lưu ý: Bài 14 cũng có **Vましょうか** với nghĩa “để tôi giúp bạn V nhé”. Bài 22 nhấn mạnh nghĩa **đề nghị cùng làm**.
