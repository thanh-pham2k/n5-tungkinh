# 2. ～たら — Nếu… thì…

## Công thức chung

```text
Thể quá khứ thông thường + ら、Mệnh đề 2
```

## Cách chia

| Loại từ          | Cách chia   | Ví dụ   |
| ---------------- | ----------- | ------- |
| Động từ          | Vた + ら      | 食べたら    |
| Động từ phủ định | Vなかった + ら   | 食べなかったら |
| Tính từ い        | Bỏ い + かったら | 暑かったら   |
| Tính từ な        | Bỏ な + だったら | 暇だったら   |
| Danh từ          | N + だったら    | 雨だったら   |

## Nghĩa

**Nếu mệnh đề 1 xảy ra thì mệnh đề 2 xảy ra.**

Ví dụ:

| Câu                  | Nghĩa                                 |
| -------------------- | ------------------------------------- |
| 暑かったら、エアコンをつけてください。  | Nếu nóng thì hãy bật điều hòa.        |
| 好きだったら、言ってください。      | Nếu thích thì hãy nói ra.             |
| 雨だったら、休んでもいいです。      | Nếu trời mưa thì nghỉ cũng được.      |
| お金があったら、旅行したいです。     | Nếu có tiền, tôi muốn đi du lịch.     |
| わからなかったら、先生に聞いてください。 | Nếu không hiểu thì hãy hỏi giáo viên. |

# 3. ～たら、どうしますか — Nếu… thì làm thế nào?

## Công thức

```text
Thể quá khứ thông thường + ら、どうしますか。
```

## Nghĩa

**Nếu tình huống đó xảy ra thì bạn sẽ làm gì / làm thế nào?**

Ví dụ:

| Câu hỏi                      | Nghĩa                                              |
| ---------------------------- | -------------------------------------------------- |
| スマホがなかったら、どうしますか。            | Nếu không có điện thoại thông minh thì bạn làm gì? |
| 勉強しているとき、眠かったら、どうしますか。       | Khi đang học, nếu buồn ngủ thì làm gì?             |
| 旅行しているとき、パスポートをなくしたら、どうしますか。 | Khi đang du lịch, nếu mất hộ chiếu thì làm gì?     |
| 地震があったら、どうしますか。              | Nếu có động đất thì làm gì?                        |

Ví dụ trả lời:

| Câu       | Nghĩa                       |
| --------- | --------------------------- |
| 子どもと遊びます。 | Tôi sẽ chơi với con.        |
| 寝ます。      | Tôi sẽ ngủ.                 |
| 大使館へ行きます。 | Tôi sẽ đi đến đại sứ quán.  |
| 机の下に入ります。 | Tôi sẽ chui xuống dưới bàn. |

# 4. Vたら、V2 — Sau khi V thì…

## Công thức

```text
Vた + ら、V2
```

## Nghĩa

**Sau khi V1 xảy ra / hoàn thành thì V2 xảy ra.**

Ví dụ:

| Câu                | Nghĩa                                 |
| ------------------ | ------------------------------------- |
| 授業が終わったら、食事に行きます。  | Sau khi giờ học kết thúc, tôi đi ăn.  |
| 10時になったら、行きましょう。   | Đến 10 giờ thì cùng đi nhé.           |
| うちへ帰ったら、シャワーを浴びます。 | Sau khi về nhà, tôi đi tắm.           |
| 卒業したら、留学します。       | Sau khi tốt nghiệp, tôi sẽ đi du học. |
| 駅に着いたら、電話してください。   | Sau khi đến ga, hãy gọi điện cho tôi. |
| 60歳になったら、仕事を辞めます。  | Khi 60 tuổi, tôi sẽ nghỉ việc.        |

# 5. Phân biệt Vてから và Vたら

| Mẫu     | Trọng tâm                                | Ví dụ                |
| ------- | ---------------------------------------- | -------------------- |
| Vてから、V2 | Nhấn mạnh V1 xong hẳn rồi mới làm V2     | レポートを書いてから、ごはんを食べます。 |
| Vたら、V2  | V1 là thời điểm / điều kiện để V2 xảy ra | 7時になったら、ごはんを食べます。    |

## Hiểu nhanh

```text
Vてから = sau khi làm xong V1 rồi mới V2
Vたら = khi / sau khi điều kiện V1 xảy ra thì V2
```

Ví dụ:

| Câu                  | Sắc thái                           |
| -------------------- | ---------------------------------- |
| レポートを書いてから、ごはんを食べます。 | Phải viết báo cáo xong rồi mới ăn. |
| 7時になったら、ごはんを食べます。    | Đến 7 giờ thì ăn cơm.              |
| 卒業してから、結婚します。        | Tốt nghiệp xong rồi mới kết hôn.   |
| 25歳になったら、結婚します。      | Đến 25 tuổi thì kết hôn.           |

---

# 6. ～ても / ～でも — Dù… thì cũng…

## Công thức

| Loại từ          | Cách chia  | Ví dụ  |
| ---------------- | ---------- | ------ |
| Động từ          | Vて + も     | 行っても   |
| Động từ phủ định | Vなくても      | 行かなくても |
| Tính từ い        | Bỏ い + くても | 暑くても   |
| Tính từ な        | Bỏ な + でも  | 便利でも   |
| Danh từ          | N + でも     | 雨でも    |

## Nghĩa

**Cho dù… thì cũng…**

Dùng khi kết quả ở mệnh đề 2 **trái với suy nghĩ thông thường** hoặc thể hiện **quyết tâm / bất chấp điều kiện**.

Ví dụ:

| Câu                | Nghĩa                                                 |
| ------------------ | ----------------------------------------------------- |
| お金がなくても、毎日楽しいです。   | Dù không có tiền, mỗi ngày vẫn vui.                   |
| 休みでも、早く起きます。       | Dù là ngày nghỉ, tôi vẫn dậy sớm.                     |
| デザインがきれいでも、買いません。  | Dù thiết kế đẹp, tôi cũng không mua.                  |
| 雨でも、友達と遊びに行きます。    | Dù trời mưa, tôi vẫn đi chơi với bạn.                 |
| 冬でも、アイスクリームを食べます。  | Dù mùa đông, tôi vẫn ăn kem.                          |
| 有名人になっても、結婚しません。   | Dù trở thành người nổi tiếng, tôi cũng không kết hôn. |
| 眠くても、勉強しなければなりません。 | Dù buồn ngủ, tôi vẫn phải học.                        |
| 大変でも、頑張ります。        | Dù vất vả, tôi vẫn cố gắng.                           |
