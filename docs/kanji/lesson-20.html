Kanji N5 – Bài 20 — 8 chữ Hán: 天、赤、青、白、黒、色、魚、犬

Bài này dạy 8 chữ Hán/Kanji N5 bài cuối cùng về trời, các màu sắc (đỏ, xanh, trắng, đen), màu sắc chung, cá và chó. Nội dung gồm âm Hán Việt, cách đọc Hiragana, nghĩa tiếng Việt và cách nhớ nhanh.

Kanji Âm Hán Việt Hiragana Romaji Nghĩa
天 Thiên てん ten trời
赤 Xích あか / あかい aka / akai màu đỏ
青 Thanh あお / あおい ao / aoi màu xanh
白 Bạch しろ / しろい shiro / shiroi màu trắng
黒 Hắc くろ / くろい kuro / kuroi màu đen
色 Sắc いろ iro màu sắc
魚 Ngư さかな sakana cá
犬 Khuyển いぬ inu chó

Cách nhớ nhanh

Kanji Cách nhớ ngắn
天 大 thêm 1 nét trên đầu → trời
赤 Lửa làm mọi thứ đỏ lên → màu đỏ
青 Trăng thanh gió mát → màu xanh
白 Tòa nhà màu trắng → màu trắng
黒 Ruộng bị đốt lâu → thành màu đen
色 Bản đồ/bát quái có màu → màu sắc
魚 Hình con cá
犬 大 thêm 1 chấm → con chó

Từ vựng ví dụ

Từ Hiragana Romaji Nghĩa
天気 てんき tenki thời tiết
赤 あか aka màu đỏ
赤い あかい akai đỏ
青 あお ao màu xanh
青い あおい aoi xanh
白 しろ shiro màu trắng
白い しろい shiroi trắng
黒 くろ kuro màu đen
黒い くろい kuroi đen
色 いろ iro màu sắc
魚 さかな sakana cá
犬 いぬ inu chó

Ghi nhớ thêm: 天気 = thời tiết, 赤い花 = hoa đỏ, 青い空 = bầu trời xanh, 白い犬 = chó trắng, 黒い車 = xe đen, 魚を食べます = ăn cá, 犬がいます = có chó.

Ghi nhớ nhanh 8 chữ Hán

Kanji Hiragana Romaji Nghĩa Ví dụ N5
天 てん ten trời 天気 = てんき / tenki = thời tiết
赤 あか / あかい aka / akai đỏ 赤い花 = あかいはな / akai hana = hoa đỏ
青 あお / あおい ao / aoi xanh 青い空 = あおいそら / aoi sora = trời xanh
白 しろ / しろい shiro / shiroi trắng 白い犬 = しろいいぬ / shiroi inu = chó trắng
黒 くろ / くろい kuro / kuroi đen 黒い車 = くろいくるま / kuroi kuruma = xe đen
色 いろ iro màu sắc 色 = いろ / iro = màu sắc
魚 さかな sakana cá 魚を食べます = さかなをたべます / sakana o tabemasu
犬 いぬ inu chó 犬がいます = いぬがいます / inu ga imasu
