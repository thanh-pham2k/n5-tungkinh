Kanji N5 – Bài 11 — 8 chữ Hán: 大、小、高、安、新、古、元、気

Bài này dạy 8 chữ Hán/Kanji N5 về kích cỡ (to, nhỏ), giá trị (cao, rẻ), tính chất (mới, cũ), gốc/nguyên và khí/tâm trạng. Nội dung gồm âm Hán Việt, cách đọc Hiragana, nghĩa tiếng Việt và cách nhớ nhanh.

Kanji Âm Hán Việt Hiragana Romaji Nghĩa
大 Đại おおきい / だい ookii / dai to / lớn
小 Tiểu ちいさい / しょう chiisai / shou nhỏ
高 Cao たかい takai cao / đắt
安 An やすい yasui rẻ / an toàn
新 Tân あたらしい atarashii mới
古 Cổ ふるい furui cũ
元 Nguyên げん gen gốc / nguyên / ban đầu
気 Khí き ki khí / tinh thần / tâm trạng

Cách nhớ nhanh

Kanji Cách nhớ ngắn
大 Người dang tay → to lớn
小 Người co lại → nhỏ bé
高 Tòa nhà cao
安 Người phụ nữ trong nhà → an toàn / yên ổn
新 Chặt cây để làm cái mới
古 Câu chuyện nói nhiều lần → trở nên cũ
元 Nguyên bản / nguyên vẹn
気 Hơi thở / không khí / tâm trạng

Từ vựng ví dụ

Từ Hiragana Romaji Nghĩa
大きい おおきい ookii to / lớn
大学 だいがく daigaku đại học
小さい ちいさい chiisai nhỏ
小学校 しょうがっこう shougakkou trường tiểu học
高い たかい takai cao / đắt
安い やすい yasui rẻ
新しい あたらしい atarashii mới
古い ふるい furui cũ
元気 げんき genki khỏe / khỏe mạnh
気持ち きもち kimochi cảm giác / tâm trạng

Ghi nhớ thêm: 大学 = đại học, 小学校 = tiểu học, 元気 = khỏe, 気持ち = cảm giác.

Ghi nhớ nhanh 8 chữ Hán

Kanji Hiragana Romaji Nghĩa Ví dụ N5
大 おおきい / だい ookii / dai to / lớn 大学 = だいがく / daigaku = đại học
小 ちいさい / しょう chiisai / shou nhỏ 小学校 = しょうがっこう / shougakkou = trường tiểu học
高 たかい takai cao / đắt 高い時計 = たかいとけい / takai tokei = đồng hồ đắt
安 やすい yasui rẻ / an toàn 安いかばん = やすいかばん / yasui kaban = cái cặp rẻ
新 あたらしい atarashii mới 新しい本 = あたらしいほん / atarashii hon = sách mới
古 ふるい furui cũ 古いテレビ = ふるいテレビ / furui terebi = TV cũ
元 げん gen gốc / nguyên / ban đầu 元気 = げんき / genki = khỏe mạnh
気 き ki khí / tinh thần / tâm trạng 気持ち = きもち / kimochi = cảm giác
