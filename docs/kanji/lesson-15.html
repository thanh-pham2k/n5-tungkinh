Kanji N5 – Bài 15 — 8 chữ Hán: 休、走、起、貝、買、売、読、書

Bài này dạy 8 chữ Hán/Kanji N5 về hành động hàng ngày (nghỉ ngơi, chạy, thức dậy), đồ vật cổ (vỏ sò) và giao dịch (mua, bán, đọc, viết). Nội dung gồm âm Hán Việt, cách đọc Hiragana, nghĩa tiếng Việt và cách nhớ nhanh.

Kanji Âm Hán Việt Hiragana Nghĩa
休 Hưu やすむ nghỉ ngơi
走 Tẩu はしる chạy
起 Khởi おきる thức dậy / xảy ra
貝 Bối かい vỏ sò
買 Mãi かう mua
売 Mại うる bán
読 Độc よむ đọc
書 Thư かく viết

Cách nhớ nhanh

Kanji Cách nhớ ngắn
休 人 + 木 → người tựa vào cây để nghỉ
走 Người chạy trên mặt đất
起 Người thức dậy rồi chạy
貝 Hình vỏ sò, xưa dùng như tiền
買 Dùng vỏ sò/tiền để mua
売 Người rao bán hàng
読 言 + 売 → đọc lời/đọc chữ ra
書 Tay cầm bút viết

Từ vựng ví dụ

Từ Hiragana Nghĩa
休みます やすみます nghỉ
休み やすみ ngày nghỉ
走ります はしります chạy
起きます おきます thức dậy
貝 かい vỏ sò
買います かいます mua
売ります うります bán
読みます よみます đọc
書きます かきます viết
本を読みます ほんをよみます đọc sách
字を書きます じをかきます viết chữ

Ghi nhớ thêm: 休みます = nghỉ, 走ります = chạy, 起きます = thức dậy, 買います = mua, 売ります = bán, 読みます = đọc, 書きます = viết, 本を読みます = đọc sách, 字を書きます = viết chữ.

Ghi nhớ nhanh 8 chữ Hán

Kanji Hiragana Nghĩa Ví dụ N5
休 やすむ nghỉ ngơi 休みます = やすみます / yasumimasu = nghỉ
走 はしる chạy 走ります = はしります / hashirimasu = chạy
起 おきる thức dậy / xảy ra 起きます = おきます / okimasu = thức dậy
貝 かい vỏ sò 貝 = かい / kai = vỏ sò
買 かう mua 買います = かいます / kaimasu = mua
売 うる bán 売ります = うります / urimasu = bán
読 よむ đọc 読みます = よみます / yomimasu = đọc
書 かく viết 書きます = かきます / kakimasu = viết
