Kanji N5 – Bài 13 — 8 chữ Hán: 入、出、市、町、村、雨、電、車

Bài này dạy 8 chữ Hán/Kanji N5 về hướng đi (vào, ra), đơn vị địa phương (thành phố, thị trấn, làng), thiên nhiên (mưa) và cơ sở hạ tầng (điện, xe). Nội dung gồm âm Hán Việt, cách đọc Hiragana, nghĩa tiếng Việt và cách nhớ nhanh.

Kanji Âm Hán Việt Hiragana Nghĩa
入 Nhập はいる vào
出 Xuất でる ra / xuất hiện
市 Thị し thành phố / chợ
町 Đinh まち thị trấn / khu phố
村 Thôn むら làng
雨 Vũ あめ mưa
電 Điện でん điện
車 Xa くるま xe

Cách nhớ nhanh

Kanji Cách nhớ ngắn
入 Người chui vào lều → vào
出 Mũi tên hướng ra ngoài → ra
市 Chợ Bến Thành / thành phố → 市
町 田 + 丁 → thị trấn có ruộng và đường
村 木 + 寸 → sau cái cây là ngôi làng
雨 Hình cơn mưa rơi
電 雨 + điện/sấm sét → điện
車 Hình chiếc xe nhìn từ trên xuống

Từ vựng ví dụ

Từ Hiragana Nghĩa
入ります はいります vào
出ます でます ra
市 し thành phố
ホーチミン市 ホーチミンし Thành phố Hồ Chí Minh
町 まち thị trấn / khu phố
村 むら làng
雨 あめ mưa
雨が降ります あめがふります trời mưa
電気 でんき điện
電車 でんしゃ tàu điện
車 くるま xe hơi / xe

Ghi nhớ thêm: 入ります = vào, 出ます = ra, 雨が降ります = trời mưa, 電気 = điện, 電車 = tàu điện, 車で行きます = đi bằng xe.

Ghi nhớ nhanh 8 chữ Hán

Kanji Hiragana Nghĩa Ví dụ N5
入 はいる vào 入ります = はいります / hairimasu = vào
出 でる ra / xuất hiện 出ます = でます / demasu = ra
市 し thành phố / chợ ホーチミン市 = ホーチミンし / Hoochimin-shi
町 まち thị trấn / khu phố 町 = まち / machi = khu phố
村 むら làng 村 = むら / mura = làng
雨 あめ mưa 雨が降ります = あめがふります / ame ga furimasu
電 でん điện 電気 = でんき / denki = điện
車 くるま xe 車で行きます = くるまでいきます / kuruma de ikimasu
