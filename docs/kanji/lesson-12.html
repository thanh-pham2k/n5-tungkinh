Kanji N5 – Bài 12 — 8 chữ Hán: 多、少、広、早、長、明、好、友

Bài này dạy 8 chữ Hán/Kanji N5 về số lượng (nhiều, ít), không gian (rộng), thời gian/tốc độ (sớm, nhanh), kích thước (dài), ánh sáng (sáng), cảm xúc (thích) và quan hệ (bạn bè). Nội dung gồm âm Hán Việt, cách đọc Hiragana, nghĩa tiếng Việt và cách nhớ nhanh.

Kanji Âm Hán Việt Hiragana Nghĩa
多 Đa おおい nhiều
少 Thiểu すくない ít
広 Quảng ひろい rộng
早 Tảo はやい sớm / nhanh
長 Trường ながい dài
明 Minh あかるい sáng
好 Hảo すき thích
友 Hữu とも bạn bè

Cách nhớ nhanh

Kanji Cách nhớ ngắn
多 Nhiều chữ タ → nhiều
少 小 thêm nét → ít / thiểu số
広 Mái nhà rộng → rộng
早 Chủ nhật ngủ tới 10 giờ vẫn còn sớm
長 Tóc/chân dài → dài
明 日 + 月 → mặt trời + mặt trăng = sáng
好 女 + 子 → phụ nữ thích trẻ con
友 Hai bàn tay nắm nhau → bạn bè

Từ vựng ví dụ

Từ Hiragana Nghĩa
多い おおい nhiều
少ない すくない ít
広い ひろい rộng
早い はやい sớm / nhanh
長い ながい dài
明るい あかるい sáng
好き すき thích
友だち ともだち bạn bè
広い部屋 ひろいへや căn phòng rộng
長い川 ながいかわ con sông dài
明るい部屋 あかるいへや căn phòng sáng

Ghi nhớ thêm: 人が多い = có nhiều người, 水が少ない = có ít nước, 広い部屋 = phòng rộng, 早く行きます = đi sớm/nhanh, 長い川 = sông dài, 明るい部屋 = phòng sáng, 友だち = bạn bè.

Ghi nhớ nhanh 8 chữ Hán

Kanji Hiragana Nghĩa Ví dụ N5
多 おおい nhiều 人が多い = ひとがおおい / hito ga ooi = nhiều người
少 すくない ít 水が少ない = みずがすくない / mizu ga sukunai = ít nước
広 ひろい rộng 広い部屋 = ひろいへや / hiroi heya = phòng rộng
早 はやい sớm / nhanh 早く行きます = はやくいきます / hayaku ikimasu = đi sớm / nhanh
長 ながい dài 長い川 = ながいかわ / nagai kawa = sông dài
明 あかるい sáng 明るい部屋 = あかるいへや / akarui heya = phòng sáng
好 すき thích 好きです = すきです / suki desu = thích
友 とも bạn bè 友だち = ともだち / tomodachi = bạn bè
