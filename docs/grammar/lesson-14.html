Minna no Nihongo Bài 14: thể て, Vてください, Vています, Vましょうか

Trọng tâm bài 14 là cách chia thể て và dùng thể này để yêu cầu, diễn tả hành động đang diễn ra, đề nghị giúp đỡ, hoặc mô tả hiện tượng tự nhiên.

Nội dung Ý nghĩa ngắn gọn Ví dụ
Thể て Dạng chia quan trọng của động từ たべます → たべて
Vてください Hãy / xin hãy làm gì まってください
Vています Đang làm gì たべています
Vましょうか Để tôi làm giúp nhé? もちましょうか
NがVています Mô tả hiện tượng tự nhiên / sự việc khách quan あめが ふっています

Cách chia thể て

Nhóm Quy tắc Ví dụ
Nhóm 1 い・ち・り → って かいます → かって
Nhóm 1 み・び・に → んで のみます → のんで
Nhóm 1 き → いて / ぎ → いで / し → して かきます → かいて、いそぎます → いそいで、はなします → はなして
Đặc biệt いきます → いって えきへ いってください。
Nhóm 2 Bỏ ます + て たべます → たべて、みます → みて、おきます → おきて
Nhóm 3 します → して、きます → きて べんきょうします → べんきょうして

Mẫu câu chính

Mẫu Nghĩa Ví dụ
Vて + ください Hãy / xin hãy làm gì なまえを かいてください。 = Hãy viết tên.
Vて + います Đang làm gì にほんごを べんきょうしています。 = Đang học tiếng Nhật.
Vます bỏ ます + ましょうか Để tôi làm giúp nhé? にもつを もちましょうか。 = Để tôi mang hành lý giúp nhé?
N が Vています Mô tả hiện tượng tự nhiên / sự việc khách quan あめが ふっています。 = Trời đang mưa.
