Minna no Nihongo Bài 3: ここ・そこ・あそこ, どこ, いくら

Bài 3 học cách hỏi và nói vị trí, dùng cách nói lịch sự こちら・そちら・あちら・どちら, hỏi xuất xứ / hãng bằng どこの, hỏi công ty / trường học bằng どちら, và hỏi giá bằng いくら.

Mẫu Nghĩa Ví dụ
ここ / そこ / あそこ chỗ này / chỗ đó / chỗ kia ここは トイレです。 = Đây là nhà vệ sinh.
こちら / そちら / あちら phía này / phía đó / phía kia, lịch sự hơn こちらは うけつけです。 = Đây là quầy lễ tân.
N は どこですか N ở đâu? トイレは どこですか。 = Nhà vệ sinh ở đâu?
N は địa điểm です N ở địa điểm đó かばんうりばは 3かいです。 = Quầy bán túi ở tầng 3.
どこの N N của nước / hãng nào? これは どこの ワインですか。
N は いくらですか N giá bao nhiêu? このほんは いくらですか。

ここ・そこ・あそこ và cách nói lịch sự

Thông thường Lịch sự Nghĩa / cách dùng
ここ こちら chỗ này / phía này, gần người nói
そこ そちら chỗ đó / phía đó, gần người nghe
あそこ あちら chỗ kia / phía kia, xa cả hai
どこ どちら ở đâu / phía nào. どちら lịch sự hơn.

Hỏi và trả lời vị trí

Mẫu Cách dùng Ví dụ
N は どこですか。 Hỏi vị trí của người, vật, địa điểm. エレベーターは どこですか。 = Thang máy ở đâu?
N は どちらですか。 Hỏi vị trí lịch sự hơn. トイレは どちらですか。 = Nhà vệ sinh ở đâu ạ?
N は địa điểm です。 Trả lời N ở địa điểm đó. トイレは あそこです。 = Nhà vệ sinh ở đằng kia.
N は tầng です。 N ở tầng nào. でんわは 2かいです。 = Điện thoại ở tầng 2.

どこの, おくには どちらですか, và どちらですか

Mẫu Nghĩa Ví dụ
どこの N ですか。 N của nước / hãng nào? それは どこの カメラですか。 = Đó là máy ảnh của hãng nào?
Nước / hãng + の + sản phẩm です。 Cách trả lời xuất xứ / hãng. フランスの ワインです。 = Là rượu vang của Pháp.
おくには どちらですか。 Bạn đến từ nước nào? / Nước của bạn là nước nào? ベトナムです。 = Việt Nam.
かいしゃ / がっこう は どちらですか。 Hỏi tên công ty / trường học, không phải hỏi vị trí. かいしゃは どちらですか。 IMCです。

いくら: hỏi giá tiền

Mẫu Nghĩa Ví dụ
N は いくらですか。 N giá bao nhiêu? このとけいは いくらですか。 = Đồng hồ này bao nhiêu tiền?
Số tiền + です。 Trả lời giá tiền. 500えんです。 = 500 yên.
えん / さい / かい / ばん yên / tuổi / tầng / số thứ tự 3,000えんです。 = 3.000 yên.
