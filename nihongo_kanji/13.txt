## TL;DR

Bài này là **Kanji N5 – Bài 13**, dạy **8 chữ Hán/Kanji**:
**入、出、市、町、村、雨、電、車**.
Nội dung chính gồm: **vào, ra, thành phố/chợ, thị trấn, làng, mưa, điện, xe**. 

---

# 1. Tổng hợp chữ Hán trong bài

| Kanji | Âm Hán Việt | Hiragana | Romaji | Nghĩa              |
| ----- | ----------- | -------- | ------ | ------------------ |
| 入     | Nhập        | はいる      | hairu  | vào                |
| 出     | Xuất        | でる       | deru   | ra / xuất hiện     |
| 市     | Thị         | し        | shi    | thành phố / chợ    |
| 町     | Đinh        | まち       | machi  | thị trấn / khu phố |
| 村     | Thôn        | むら       | mura   | làng               |
| 雨     | Vũ          | あめ       | ame    | mưa                |
| 電     | Điện        | でん       | den    | điện               |
| 車     | Xa          | くるま      | kuruma | xe                 |

---

# 2. Ghi nhớ nhanh

| Kanji | Cách nhớ ngắn                      |
| ----- | ---------------------------------- |
| 入     | Người chui vào lều → vào           |
| 出     | Mũi tên hướng ra ngoài → ra        |
| 市     | Chợ Bến Thành / thành phố → 市      |
| 町     | 田 + 丁 → thị trấn có ruộng và đường |
| 村     | 木 + 寸 → sau cái cây là ngôi làng   |
| 雨     | Hình cơn mưa rơi                   |
| 電     | 雨 + điện/sấm sét → điện            |
| 車     | Hình chiếc xe nhìn từ trên xuống   |
