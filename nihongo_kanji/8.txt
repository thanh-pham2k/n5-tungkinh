## TL;DR

Bài này là **Kanji N5 – Bài 8**, dạy **8 chữ Hán/Kanji**:
**何、父、母、年、去、毎、王、国**.
Nội dung chính gồm: **cái gì, cha, mẹ, năm, quá khứ/đi, mỗi, vua, đất nước**. 

---

# 1. Tổng hợp chữ Hán trong bài

| Kanji（かな） | Âm Hán Việt | Nghĩa |
| --- | --- | --- |
| 何（なに / なん） | Hà | cái gì |
| 父（ちち） | Phụ | cha / bố |
| 母（はは） | Mẫu | mẹ |
| 年（とし / ねん） | Niên | năm / tuổi |
| 去（きょ / さる） | Khứ | quá khứ / rời đi |
| 毎（まい） | Mỗi | mỗi |
| 王（おう） | Vương | vua |
| 国（くに / こく） | Quốc | đất nước |

---

# 2. Ghi nhớ nhanh

| Kanji | Cách nhớ ngắn                        |
| ----- | ------------------------------------ |
| 何     | Người đang hỏi: “cái gì?”            |
| 父     | Hình người cha                       |
| 母     | Hình người mẹ ôm con                 |
| 年     | Một năm thu hoạch một lần            |
| 去     | Chuyện đã qua → quá khứ              |
| 毎     | Mẹ che chở mỗi ngày                  |
| 王     | Vua đội vương miện                   |
| 国     | Vua/ngọc ở trong lãnh thổ → đất nước |
