## TL;DR

Bài này là **Kanji N5 – Bài 15**, dạy **8 chữ Hán/Kanji**:
**休、走、起、貝、買、売、読、書**.
Nội dung chính gồm: **nghỉ ngơi, chạy, thức dậy, vỏ sò/tiền, mua, bán, đọc, viết**. 

---

# 1. Tổng hợp chữ Hán trong bài

| Kanji | Âm Hán Việt | Hiragana | Romaji  | Nghĩa             |
| ----- | ----------- | -------- | ------- | ----------------- |
| 休     | Hưu         | やすむ      | yasumu  | nghỉ ngơi         |
| 走     | Tẩu         | はしる      | hashiru | chạy              |
| 起     | Khởi        | おきる      | okiru   | thức dậy / xảy ra |
| 貝     | Bối         | かい       | kai     | vỏ sò             |
| 買     | Mãi         | かう       | kau     | mua               |
| 売     | Mại         | うる       | uru     | bán               |
| 読     | Độc         | よむ       | yomu    | đọc               |
| 書     | Thư         | かく       | kaku    | viết              |

---

# 2. Ghi nhớ nhanh

| Kanji | Cách nhớ ngắn                     |
| ----- | --------------------------------- |
| 休     | 人 + 木 → người tựa vào cây để nghỉ |
| 走     | Người chạy trên mặt đất           |
| 起     | Người thức dậy rồi chạy           |
| 貝     | Hình vỏ sò, xưa dùng như tiền     |
| 買     | Dùng vỏ sò/tiền để mua            |
| 売     | Người rao bán hàng                |
| 読     | 言 + 売 → đọc lời/đọc chữ ra        |
| 書     | Tay cầm bút viết                  |
