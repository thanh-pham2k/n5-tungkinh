## TL;DR

Bài này là **Kanji N5 – Bài 7**, dạy **8 chữ Hán/Kanji**:
**田、力、男、女、子、学、生、先**.
Nội dung chính gồm: **ruộng, sức lực, nam/nữ, trẻ em, học, sinh, trước/tiên sinh**. 

---

# 1. Tổng hợp chữ Hán trong bài

| Kanji（かな） | Âm Hán Việt | Nghĩa |
| --- | --- | --- |
| 田（た） | Điền | ruộng |
| 力（ちから） | Lực | sức lực |
| 男（おとこ） | Nam | đàn ông / nam |
| 女（おんな） | Nữ | phụ nữ / nữ |
| 子（こ） | Tử | đứa trẻ / con |
| 学（がく） | Học | học |
| 生（せい / うまれる） | Sinh | sinh / sống / học sinh |
| 先（さき / せん） | Tiên | trước |

---

# 2. Ghi nhớ nhanh

| Kanji | Cách nhớ ngắn                                |
| ----- | -------------------------------------------- |
| 田     | Mảnh ruộng được chia ô                       |
| 力     | Người có cơ bắp → sức lực                    |
| 男     | 田 + 力 → người đàn ông làm ruộng bằng sức lực |
| 女     | Hình dáng người phụ nữ                       |
| 子     | Hình đứa trẻ                                 |
| 学     | Đứa trẻ đi học                               |
| 生     | Cây mọc lên → sinh sống                      |
| 先     | Người đi trước → trước / tiên                |
