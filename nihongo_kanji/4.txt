## TL;DR

Bài này là **Kanji N5 – Bài 4**, dạy **8 chữ Hán/Kanji**:
**本、人、今、寺、時、半、刀、分**.
Nội dung chính gồm: **âm Hán Việt, nghĩa, cách đọc, cách liên tưởng và cách viết**. 

---

# 1. Tổng hợp chữ Hán trong bài

| Kanji | Âm Hán Việt | Hiragana     | Romaji           | Nghĩa              |
| ----- | ----------- | ------------ | ---------------- | ------------------ |
| 本     | Bản         | ほん           | hon              | sách / gốc / bản   |
| 人     | Nhân        | ひと / じん / にん | hito / jin / nin | người              |
| 今     | Kim         | いま           | ima              | bây giờ / hiện tại |
| 寺     | Tự          | てら           | tera             | chùa               |
| 時     | Thời        | じ            | ji               | giờ / thời gian    |
| 半     | Bán         | はん           | han              | một nửa            |
| 刀     | Đao         | かたな          | katana           | dao / kiếm         |
| 分     | Phân        | ふん / ぷん      | fun / pun        | phút / phần / chia |

---

# 2. Ghi nhớ nhanh

| Kanji | Cách nhớ ngắn                           |
| ----- | --------------------------------------- |
| 本     | 木 thêm 1 nét → cây thành sách           |
| 人     | Hình người đang đứng                    |
| 今     | Người dưới mái nhà → hiện tại           |
| 寺     | Đất + đo lường → chùa                   |
| 時     | 日 + 寺 → nhìn mặt trời ở chùa để xem giờ |
| 半     | Chia thành 2 nửa                        |
| 刀     | Hình con dao / kiếm                     |
| 分     | 八 + 刀 → dao chia ra nhiều phần          |
