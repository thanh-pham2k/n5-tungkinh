## TL;DR

Bài này là **Kanji N5 – Bài 19**, dạy **8 chữ Hán/Kanji**:
**山、川、林、森、空、海、化、花**.
Nội dung chính gồm: **núi, sông, rừng nhỏ, rừng lớn, bầu trời, biển, biến hóa, hoa**. 

---

# 1. Tổng hợp chữ Hán trong bài

| Kanji | Âm Hán Việt | Hiragana | Romaji  | Nghĩa               |
| ----- | ----------- | -------- | ------- | ------------------- |
| 山     | Sơn         | やま       | yama    | núi                 |
| 川     | Xuyên       | かわ       | kawa    | sông                |
| 林     | Lâm         | はやし      | hayashi | rừng nhỏ            |
| 森     | Sâm         | もり       | mori    | rừng lớn            |
| 空     | Không       | そら       | sora    | bầu trời            |
| 海     | Hải         | うみ       | umi     | biển                |
| 化     | Hóa         | か        | ka      | biến hóa / thay đổi |
| 花     | Hoa         | はな       | hana    | hoa                 |

---

# 2. Ghi nhớ nhanh

| Kanji | Cách nhớ ngắn                             |
| ----- | ----------------------------------------- |
| 山     | Hình ngọn núi                             |
| 川     | Hình dòng sông                            |
| 林     | 2 cây 木 + 木 → rừng nhỏ                    |
| 森     | 3 cây 木 + 木 + 木 → rừng lớn                |
| 空     | Đo không khí bên ngoài → bầu trời         |
| 海     | 氵 + 毎 → mỗi ngày có nước → biển           |
| 化     | Người đứng + người ngồi → biến đổi tư thế |
| 花     | Cỏ cây biến hóa thành hoa                 |
