## TL;DR

Bài này là **Kanji N5 – Bài 10**, dạy **8 chữ Hán/Kanji**:
**耳、聞、言、話、立、待、周、週**.
Nội dung chính gồm: **tai, nghe/hỏi, lời nói, nói chuyện, đứng/lập, chờ, xung quanh, tuần**. 

---

# 1. Tổng hợp chữ Hán trong bài

| Kanji（かな） | Âm Hán Việt | Nghĩa |
| --- | --- | --- |
| 耳（みみ） | Nhĩ | tai |
| 聞（きく） | Văn | nghe / hỏi |
| 言（いう） | Ngôn | nói |
| 話（はなす） | Thoại | nói chuyện |
| 立（たつ） | Lập | đứng / lập |
| 待（まつ） | Đãi | chờ |
| 周（まわり） | Chu | xung quanh |
| 週（しゅう） | Chu | tuần |

---

# 2. Ghi nhớ nhanh

| Kanji | Cách nhớ ngắn                    |
| ----- | -------------------------------- |
| 耳     | Hình cái tai                     |
| 聞     | 門 + 耳 → tai nghe qua cổng        |
| 言     | Cái miệng tạo ra lời nói         |
| 話     | 言 + 舌/口 → dùng lời để nói chuyện |
| 立     | Người đứng thẳng                 |
| 待     | Người đứng trên đường để chờ     |
| 周     | Bao quanh một vùng               |
| 週     | Đi một vòng đủ 1 tuần            |

---
