## TL;DR

Bài này là **Kanji N5 – Bài 3**, dạy **8 chữ Hán/Kanji** liên quan đến **mặt trời, mặt trăng, ngũ hành và ngày trong tuần**:
**日、月、火、水、木、土、金、曜**. 

---

# 1. Tổng hợp chữ Hán trong bài

| Kanji（かな） | Âm Hán Việt | Nghĩa chính |
| --- | --- | --- |
| 日（ひ / にち） | Nhật | mặt trời / ngày |
| 月（つき / げつ / がつ） | Nguyệt | mặt trăng / tháng |
| 火（ひ / か） | Hỏa | lửa |
| 水（みず / すい） | Thủy | nước |
| 木（き / もく） | Mộc | cây |
| 土（つち / ど） | Thổ | đất |
| 金（かね / きん） | Kim | tiền / vàng / kim loại |
| 曜（よう） | Diệu | ngày trong tuần |

---

# 2. Ghi nhớ nhanh

| Kanji | Cách nhớ ngắn                      |
| ----- | ---------------------------------- |
| 日     | Mặt trời → ngày                    |
| 月     | Mặt trăng → tháng                  |
| 火     | Hình ngọn lửa                      |
| 水     | Hình dòng nước                     |
| 木     | Hình cái cây                       |
| 土     | Cây mọc từ đất                     |
| 金     | Kim loại / vàng / tiền             |
| 曜     | Dùng trong các ngày trong tuần: 曜日 |

---

# 3. Ứng dụng vào thứ trong tuần

| Từ（かな） | Nghĩa |
| --- | --- |
| 日曜日（にちようび） | Chủ nhật |
| 月曜日（げつようび） | Thứ hai |
| 火曜日（かようび） | Thứ ba |
| 水曜日（すいようび） | Thứ tư |
| 木曜日（もくようび） | Thứ năm |
| 金曜日（きんようび） | Thứ sáu |
| 土曜日（どようび） | Thứ bảy |
