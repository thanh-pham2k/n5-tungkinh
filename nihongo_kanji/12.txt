## TL;DR

Bài này là **Kanji N5 – Bài 12**, dạy **8 chữ Hán/Kanji**:
**多、少、広、早、長、明、好、友**.
Nội dung chính gồm: **nhiều, ít, rộng, sớm/nhanh, dài, sáng, thích, bạn bè**. 

---

# 1. Tổng hợp chữ Hán trong bài

| Kanji | Âm Hán Việt | Hiragana | Romaji  | Nghĩa       |
| ----- | ----------- | -------- | ------- | ----------- |
| 多     | Đa          | おおい      | ooi     | nhiều       |
| 少     | Thiểu       | すくない     | sukunai | ít          |
| 広     | Quảng       | ひろい      | hiroi   | rộng        |
| 早     | Tảo         | はやい      | hayai   | sớm / nhanh |
| 長     | Trường      | ながい      | nagai   | dài         |
| 明     | Minh        | あかるい     | akarui  | sáng        |
| 好     | Hảo         | すき       | suki    | thích       |
| 友     | Hữu         | とも       | tomo    | bạn bè      |

---

# 2. Ghi nhớ nhanh

| Kanji | Cách nhớ ngắn                       |
| ----- | ----------------------------------- |
| 多     | Nhiều chữ タ → nhiều                 |
| 少     | 小 thêm nét → ít / thiểu số          |
| 広     | Mái nhà rộng → rộng                 |
| 早     | Chủ nhật ngủ tới 10 giờ vẫn còn sớm |
| 長     | Tóc/chân dài → dài                  |
| 明     | 日 + 月 → mặt trời + mặt trăng = sáng |
| 好     | 女 + 子 → phụ nữ thích trẻ con        |
| 友     | Hai bàn tay nắm nhau → bạn bè       |
