## TL;DR

Bài này là **Kanji N5 – Bài 6**, dạy **8 chữ Hán/Kanji**:
**後、午、門、間、東、北、西、南**.
Nội dung chính gồm: **phía sau, buổi trưa, cổng, khoảng/thời gian, 4 phương hướng**. 

---

# 1. Tổng hợp chữ Hán trong bài

| Kanji | Âm Hán Việt | Hiragana | Romaji      | Nghĩa                     |
| ----- | ----------- | -------- | ----------- | ------------------------- |
| 後     | Hậu         | うしろ / ご  | ushiro / go | sau / phía sau            |
| 午     | Ngọ         | ご        | go          | buổi trưa / 12 giờ        |
| 門     | Môn         | もん       | mon         | cổng                      |
| 間     | Gian        | あいだ / かん | aida / kan  | khoảng / giữa / thời gian |
| 東     | Đông        | ひがし      | higashi     | phía Đông                 |
| 北     | Bắc         | きた       | kita        | phía Bắc                  |
| 西     | Tây         | にし       | nishi       | phía Tây                  |
| 南     | Nam         | みなみ      | minami      | phía Nam                  |

---

# 2. Ghi nhớ nhanh

| Kanji | Cách nhớ ngắn                                        |
| ----- | ---------------------------------------------------- |
| 後     | Người quay lại nhìn phía sau                         |
| 午     | Mốc 12 giờ trưa → 午前 / 午後                            |
| 門     | Hình cái cổng                                        |
| 間     | 門 + 日 → ánh sáng/mặt trời lọt qua cổng → khoảng giữa |
| 東     | Mặt trời sau cái cây → hướng Đông                    |
| 北     | Hai người quay lưng lại → phương Bắc lạnh            |
| 西     | Hình cái giỏ / hướng Tây                             |
| 南     | Phía Nam, vùng ấm hơn, nhiều cây trái                |
