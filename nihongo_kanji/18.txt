## TL;DR

Bài này là **Kanji N5 – Bài 18**, dạy **8 chữ Hán/Kanji**:
**晩、夜、心、手、足、体、首、道**.
Nội dung chính gồm: **buổi tối, ban đêm, tim/tâm, tay, chân, cơ thể, cổ/đầu, con đường/đạo**. 

---

# 1. Tổng hợp chữ Hán trong bài

| Kanji（かな） | Âm Hán Việt | Nghĩa |
| --- | --- | --- |
| 晩（ばん） | Vãn | buổi tối |
| 夜（よる） | Dạ | đêm / ban đêm |
| 心（こころ） | Tâm | tim / tâm trí |
| 手（て） | Thủ | tay |
| 足（あし） | Túc | chân |
| 体（からだ） | Thể | cơ thể |
| 首（くび） | Thủ | cổ / đầu |
| 道（みち） | Đạo | con đường / đạo |

---

# 2. Ghi nhớ nhanh

| Kanji | Cách nhớ ngắn                  |
| ----- | ------------------------------ |
| 晩     | Mặt trời xuống → buổi tối      |
| 夜     | Đêm khuya / ban đêm            |
| 心     | Hình trái tim → tâm, cảm xúc   |
| 手     | Hình bàn tay                   |
| 足     | Miệng + chân → chân            |
| 体     | 人 + 本 → thân thể / cơ thể      |
| 首     | Phần đầu/cổ                    |
| 道     | 首 + đường đi → con đường / đạo |

---
