# 問題1 — Chọn cách đọc Hiragana

## 1. あしたは 雨ですか。

**Đáp án đúng: 2. あめ**

**Giải thích:**
雨 = **あめ** = mưa.
Câu nghĩa là: “Ngày mai trời mưa à?”

**Ngữ pháp:**
`Nですか` = có phải là N không?
`雨ですか` = trời mưa à?

**Từ vựng:**

| Từ  | Đọc | Nghĩa       |
| --- | --- | ----------- |
| 明日  | あした | ngày mai    |
| 雨   | あめ  | mưa         |
| ですか | ですか | phải không? |

**Minna nên ôn:** Bài 4, Bài 12.

**Đáp án sai:**

1. ゆき = 雪 = tuyết.
2. くもり = trời nhiều mây.
3. はれ = trời nắng.

---

## 2. きょうしつで 書いてください。

**Đáp án đúng: 3. かいて**

**Giải thích:**
書いて = **かいて** = hãy viết.
書きます → 書いて.

**Ngữ pháp:**
`Vてください` = hãy làm gì.
`場所で Vます` = làm hành động ở đâu.
→ 教室で書いてください = Hãy viết ở lớp học.

**Từ vựng:**

| Từ      | Đọc     | Nghĩa    |
| ------- | ------- | -------- |
| 教室      | きょうしつ   | lớp học  |
| 書きます    | かきます    | viết     |
| 書いてください | かいてください | hãy viết |

**Minna nên ôn:** Bài 14.

**Đáp án sai:**

1. はいて = mặc/quàng phần dưới, đi giày.
2. きいて = nghe/hỏi.
3. ひいて = kéo / chơi nhạc cụ / bị cảm.

---

## 3. しゃしんは はこの 中に あります。

**Đáp án đúng: 2. なか**

**Giải thích:**
中 = **なか** = bên trong.
Câu nghĩa là: “Ảnh ở trong hộp.”

**Ngữ pháp:**
`Nは 場所に あります` = N ở đâu.
`箱の中にあります` = ở trong hộp.

**Từ vựng:**

| Từ   | Kanji | Nghĩa  |
| ---- | ----- | ------ |
| 写真   | しゃしん  | ảnh    |
| 箱    | はこ    | hộp    |
| 中    | なか    | trong  |
| あります | あります  | có / ở |

**Minna nên ôn:** Bài 10.

**Đáp án sai:**

1. よこ = bên cạnh.
2. そと = bên ngoài.
3. そば = gần / cạnh.

---

## 4. この いすは 小さいです。

**Đáp án đúng: 3. ちいさい**

**Giải thích:**
小さい = **ちいさい** = nhỏ.
Câu nghĩa là: “Cái ghế này nhỏ.”

**Ngữ pháp:**
`Nは い形容詞です` = N thì như thế nào.
`小さいです` = nhỏ.

**Từ vựng:**

| Từ  | Đọc  | Nghĩa   |
| --- | ---- | ------- |
| いす  | いす   | ghế     |
| 小さい | ちいさい | nhỏ     |
| この  | この   | cái này |

**Minna nên ôn:** Bài 8, Bài 12.

**Đáp án sai:**

1. しいさい: sai âm.
2. しさい: sai.
3. ちさい: thiếu âm い.

## 5. あしたは 火よう日です。

**Đáp án đúng: 1. かようび**

**Giải thích:**
火よう日 = 火曜日 = **かようび** = thứ Ba.

**Ngữ pháp:**
`Nは 曜日です` = N là thứ mấy.
`明日は火曜日です` = ngày mai là thứ Ba.

**Từ vựng:**

| Từ  | Kanji | Nghĩa    |
| --- | ----- | -------- |
| 明日  | あした   | ngày mai |

## 13. この わいしゃつを ください。

**Đáp án đúng: 2. ワイシャツ**

**Giải thích:**
わいしゃつ viết Katakana là **ワイシャツ** = áo sơ mi.

**Ngữ pháp:**
`Nをください` = cho tôi N.
`このワイシャツをください` = cho tôi cái áo sơ mi này.

**Từ vựng:**

| Từ    | Nghĩa             |
| ----- | ----------------- |
| ワイシャツ | áo sơ mi          |
| この    | cái này           |
| ください  | cho tôi / xin hãy |

**Minna nên ôn:** Bài 3, Bài 7.

**Đáp án sai:**

1. ワイシャソ: sai chữ cuối, ソ khác ツ.
2. ウイシャソ: sai âm đầu và sai chữ cuối.
3. ウイシャツ: sai âm đầu, đúng là ワイ.

## 14. わたしの くには かわが おおいです。

**Giải thích:**
かわ = **川** = sông.
